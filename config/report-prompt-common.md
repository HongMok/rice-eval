# 通用评估报告 — 计算规则

> 本文档定义用户自建评估工具（非 PEP-3、C-PEP、VB-MAPP 标准量表）的报告计算规则。
> 适配任意评分体系（2级、3级、4级等），不硬编码评分选项。

---

## 1. 领域得分计算

```
领域总得分 = 该领域所有题目得分之和
领域满分 = 题目数 × 评分体系最高分
领域得分率 = 领域总得分 / 领域满分 × 100%（四舍五入取整）
```

---

## 2. 总分计算

```
总得分 = 所有领域总得分之和
总满分 = 所有领域满分之和
总得分率 = 总得分 / 总满分 × 100%
```

---

## 3. 评分分布统计

按评分体系的各等级标签，统计每个领域中各等级的题目数量。

---

## 4. 综合分析规则

| 模块 | 规则 |
|------|------|
| 概述 | 包含工具名称、总题数、领域数、总得分率 |
| 优势领域 | 得分率最高的领域 + 已达到最高分的具体项目 |
| 需关注领域 | 得分率最低的领域 + 得分为 0 的具体项目 |
| 训练建议 | 针对薄弱领域，优先从"辅助下完成"的项目入手（最近发展区原则） |

---

## 5. 确定性要求

- 同一份输入数据 → 同一份报告输出
- 不使用随机措辞
- 所有数值计算使用四舍五入
