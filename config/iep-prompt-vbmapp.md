# VB-MAPP 评估结果 → IEP 生成规则

```
=== INPUT ===

1. 个案基本信息
   - 姓名、性别、出生日期 → 计算生理年龄（月）
   - 诊断类型（档案中获取）：自闭症/语言发育迟缓/其他
   - 评估日期、评估师
   - IEP 时间范围：起止日期（如 2026-05-01 ~ 2026-07-31）→ 计算周期月数

2. VB-MAPP 评估数据（三部分）

   里程碑评估（170个里程碑，3个等级，16个领域）：每项 1=达标 / 0.5=部分达标 / 0=未达标
   ┌──────────────────┬────┬────┬────┬────┐
   │ 领域              │ L1 │ L2 │ L3 │满分│
   │ 提要求(Mand)       │  5 │  5 │  5 │ 15 │
   │ 命名(Tact)         │  5 │  5 │  5 │ 15 │
   │ 听者反应(LR)       │  5 │  5 │  5 │ 15 │
   │ 视觉配对(VP-MTS)   │  5 │  5 │  5 │ 15 │
   │ 游戏(Play)         │  5 │  5 │  - │ 10 │
   │ 社交(Social)       │  5 │  5 │  5 │ 15 │
   │ 模仿(Imitation)    │  5 │  5 │  - │ 10 │
   │ 仿说(Echoic)       │  5 │  5 │  - │ 10 │
   │ 发音(SVB)          │  5 │  - │  - │  5 │
   │ LRFFC              │  - │  5 │  5 │ 10 │
   │ 对话(IV)           │  - │  5 │  5 │ 10 │
   │ 集体技能(Group)     │  - │  5 │  5 │ 10 │
   │ 语言结构(Ling)      │  - │  5 │  5 │ 10 │
   │ 数学(Math)         │  - │  - │  5 │  5 │
   │ 阅读(Reading)      │  - │  - │  5 │  5 │
   │ 书写(Writing)      │  - │  - │  5 │  5 │
   │ 合计               │ 45 │ 60 │ 65 │170 │
   └──────────────────┴────┴────┴────┴────┘
   L1=0-18月 | L2=18-30月 | L3=30-48月

   障碍评估（24项）：每项 0~4分，越高障碍越严重
   ┌────┬──────────────────────────┐
   │ #  │ 障碍名称                  │
   │  1 │ 不恰当行为                │
   │  2 │ 教学控制薄弱/逃避教学      │
   │  3 │ 提要求的缺陷              │
   │  4 │ 命名的缺陷                │
   │  5 │ 模仿的缺陷                │
   │  6 │ 仿说的缺陷                │
   │  7 │ 视觉感知和配对的缺陷       │
   │  8 │ 听者技能的缺陷            │
   │  9 │ 对话的缺陷                │
   │ 10 │ 社会技能的缺陷            │
   │ 11 │ 依赖辅助                  │
   │ 12 │ 随意猜想反应              │
   │ 13 │ 扫视功能的缺失            │
   │ 14 │ 不能进行制约分辨          │
   │ 15 │ 泛化能力的缺失            │
   │ 16 │ 较弱或非常规的动机操作     │
   │ 17 │ 对反应的要求会减弱动机操作 │
   │ 18 │ 对强化物的依赖            │
   │ 19 │ 自我刺激                  │
   │ 20 │ 构音问题                  │
   │ 21 │ 强迫的行为                │
   │ 22 │ 过度多动的行为            │
   │ 23 │ 不能与他人进行眼神接触     │
   │ 24 │ 感觉系统防御              │
   └────┴──────────────────────────┘
   满分96，总体评级：0-24=低 | 25-48=中 | 49-72=较高 | 73-96=严重

   转衔评估（18项，3大类别）：每项 0~5分，越高准备越充分
   第一类：VB-MAPP得分和学业独立性（7项）
   第二类：学习模式（6项）
   第三类：自理、自发和自主（5项）
   满分90，总体建议：0-30=密集个训 | 31-60=个训+小组 | 61-90=可融合

3. 课包干预项目库
   - 指定课包（如 RICE1v1）中的全部干预项目
   - 每个项目含：name / territory / subdivide / age_bracket / stages[]

=== OUTPUT ===

输出：IEP 表格
┌──────────────┬──────────┬──────────┐
│ 阶梯及领域    │ 任务目标  │ 短期目标  │
│ {年龄段}-     │ {项目name}│ {阶段name}│
│ {territory}- │          │          │
│ {subdivide}  │          │          │
└──────────────┴──────────┴──────────┘

=== 推导规则 ===

一、计算各领域指标

   里程碑评估：
   领域得分率 = 领域总得分 / 领域满分 × 100%
   里程碑总分 → 发展阶段：0-45=L1(0-18月) | 46-90=L2(18-30月) | 91-170=L3(30-48月)
   领域评级：≥70%=表现良好 | 40%-69%=发展中 | <40%=需重点关注
   薄弱判定：得分率 ≤ 50% → 薄弱

   障碍评估：
   单项严重度：0=无 | 1=轻微 | 2=中等 | 3=较严重 | 4=严重
   总分评级：0-24=低 | 25-48=中 | 49-72=较高 | 73-96=严重

   转衔评估：
   单项评级：0-1=需重点关注 | 2-3=需提升 | 4-5=已具备
   总分建议：0-30=密集个训 | 31-60=个训+小组 | 61-90=可融合

二、确定优先级排序（权重：诊断类型 > 障碍 > 里程碑）

   P0：诊断相关核心缺陷
       自闭症诊断 → 社交(Social) + 提要求(Mand) 领域强制纳入
       语言发育迟缓 → 提要求(Mand) + 命名(Tact) + 听者反应(LR) 强制纳入
       → 诊断类型决定哪些领域"必须有"，不受得分率门槛限制

   P1：障碍评估得分 ≥ 3（严重障碍必须优先处理）
       → 严重障碍不解决，技能训练效果大打折扣
       → 常见：不恰当行为、教学控制薄弱、自我刺激
       → 匹配课包"学习行为"领域，无对应项目则备注"建议制定BIP"

   P2：障碍评估得分 = 2（中度障碍）
       → 在相关技能领域附加行为支持策略

   P3：提要求(Mand)领域（Sundberg核心建议）
       → 提要求是语言行为基础，直接受动机控制，训练效果最快
       → 即使其他领域更弱，Mand 也应优先或同步干预
       → 能减少因无法表达需求而产生的行为问题

   P4：里程碑得分率 ≤ 30% 且处于当前等级的领域

   P5：里程碑得分率 31%~50% 的领域

   P6：模仿(Imitation) + 仿说(Echoic)领域
       → 模仿是学习其他技能的前提

   P7：转衔评估得分 ≤ 1 的领域（仅当儿童即将转衔时）

三、生理年龄与发展阶段差异 → 调整干预策略

   VB-MAPP 用里程碑等级定位发展水平：
   当前等级 = 该领域最高通过的里程碑所在等级
   等级对应发展年龄：L1=0-18月 | L2=18-30月 | L3=30-48月

   差距 = 生理年龄 - 等级对应年龄上限（月）

   差距 ≤ 6月：基本持平，常规干预
   差距 7~12月：轻度落后，增加训练频次
   差距 13~24月：中度落后，该领域优先级提升一档
   差距 > 24月：显著落后，该领域强制纳入 IEP

   全面落后（总分 ≤ 45，停留在L1）：
   → 只选提要求(Mand) + 模仿(Imitation) + 听者反应(LR) 3个基础领域
   → 其余下个周期再纳入（Sundberg建议）

   能力不均衡（领域间差距 > 30%）：
   → 弱势领域正常纳入
   → 强势领域标注"可作为教学媒介"

四、限定干预目标数量（根据 IEP 时间范围梯度设计）

   ┌──────────┬──────┬───────┬──────────────────────────────┐
   │ 周期月数  │领域数 │总目标数│ 说明                          │
   │ 1个月    │ 1~2  │ 3~5   │ 短期集中突破，聚焦最核心问题    │
   │ 2个月    │ 2~3  │ 5~8   │ 核心领域为主，少量扩展          │
   │ 3个月    │ 3~5  │ 8~15  │ 标准周期，覆盖主要薄弱领域      │
   │ 6个月    │ 4~6  │ 12~20 │ 长周期，分阶段推进，可设中期检查 │
   │ 12个月   │ 5~8  │ 15~25 │ 年度计划，按季度分解子目标      │
   └──────────┴──────┴───────┴──────────────────────────────┘

   按发展阶段微调（在上表基础上）：
   L1(0-18月) → 领域数和目标数取下限，集中突破
   L2(18-30月) → 取中间值，逐步扩展
   L3(30-48月) → 取上限，全面发展

   每领域任务目标：2~4个（上限5）
   每目标短期阶段：1~3个
   长周期（≥6个月）梯度设计：
   → 前1/3周期：基础目标（从0.5分技能切入）
   → 中1/3周期：进阶目标（巩固+扩展）
   → 后1/3周期：泛化目标（跨情境应用）

五、匹配干预项目

   领域映射规则（评估领域 → 课包领域）：
   不写死映射表，按以下规则动态匹配课包中的干预项目：
   1. 语义匹配：评估领域名称与课包 territory/subdivide 进行语义相似度匹配
      - 优先精确匹配（如"提要求"匹配课包中含"语言表达"的项目）
      - 次选模糊匹配（如"视觉配对"匹配课包中含"视觉""配对"关键词的项目）
   2. 能力维度匹配：根据评估领域考察的核心能力维度，匹配课包中训练相同能力的项目
      - 语言输出类领域（提要求/命名/对话/语言结构）→ 匹配课包中语言表达相关项目
      - 语言理解类领域（听者反应/LRFFC）→ 匹配课包中语言理解相关项目
      - 运动类领域（游戏）→ 匹配课包中游戏/运动相关项目
      - 社交类领域（社交/集体技能）→ 匹配课包中社交相关项目
      - 认知类领域（视觉配对/模仿）→ 匹配课包中认知/模仿相关项目
      - 发音类领域（仿说/发音）→ 匹配课包中发音/构音相关项目
   3. 年龄段过滤：匹配结果按干预年龄段筛选
   4. 一个评估领域可匹配课包中多个 territory/subdivide

   年龄段匹配（按里程碑等级，不按生理年龄）：
   L1 → 1-2岁 | L2 → 2-3岁 | L3 → 3-4岁
   当前等级里程碑 ≥ 70% 通过 → 可进入下一等级年龄段
   边界处理：优先选较低年龄段中未掌握的项目

   匹配排序：
   a. 与 0.5分（部分达标）技能直接对应的干预项目（最优先）
   b. 与 0分（未达标）技能对应的干预项目中，阶段最低的
   c. 该领域该年龄段的其他干预项目（补充）

   匹配失败 → 跳过，备注"XX领域建议补充专项干预方案"
   绝对不能编造不存在的项目名称

六、0.5分项处理（类似萌芽技能）

   0.5分 = 部分达标 = 最近发展区 = IEP 核心切入点
   → 干预从当前水平继续推进，不从零开始
   → 在 IEP 中优先于 0分的项目
   → 示例：命名5M=0.5分（能命名8个物品，目标10个）→ 从8个起步训练

=== 约束 ===

- 同一份评估数据 + 同一个课包 → 同一份 IEP
- 所有项目名称和阶段名称必须与课包完全一致
- 不能编造、改写或合并课包中的项目
- ABA 术语保持准确：提要求=Mand，命名=Tact，听者反应=LR，LRFFC 等
- IEP 周期：由输入的时间范围决定，短期目标预期 2~4 周达成
- 导出文件：{姓名}个别化教育计划.docx
```
