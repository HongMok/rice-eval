# PEP-3 评估结果 → IEP 生成规则

> 本文档定义 PEP-3 评估结果如何转化为个别化教育计划（IEP）。
> 核心原则：IEP 中的所有内容必须来自课包干预项目配置表，不能自行编造。

---

## 1. IEP 表格结构

| 列 | 数据来源 | 示例 |
|----|---------|------|
| 阶梯及领域 | `{年龄段}-{一级领域}-{二级领域}` | `1-2岁-发音-韵母`、`2-3岁-沟通-语言理解` |
| 任务目标 | 干预项目名称（`intervene_project.name`） | `韵母a`、`简单句理解` |
| 短期目标 | 干预项目阶段名称（`intervene_project_stage.name`） | `诱导发/a/音`、`人物+动作/地点` |

---

## 2. 薄弱领域筛选

### 2.1 阈值

```
薄弱领域 = 功能发展得分率 ≤ 70% 的领域
```

### 2.2 排序

按得分率升序排列（最薄弱的排最前）。

### 2.3 行为领域

行为表现评级为 S（严重）的领域也纳入，优先级高于功能发展薄弱领域。

---

## 3. 领域映射

PEP-3 评估领域 → 课包干预领域的映射关系：

| PEP-3 领域 | 课包领域 |
|------------|---------|
| 认知表达(CV) | 沟通-语言表达 |
| 认知理解(CP) | 沟通-语言理解 |
| 模仿(I) | 模仿 |
| 小肌肉(FM) | 精细动作 |
| 大肌肉(GM) | 粗大动作 |
| 手眼协调(EH) | 精细动作-手眼协调 |
| 知觉(P) | 感知觉 |
| 情感关系(R) | 社交情绪 |
| 游戏及兴趣(M) | 社交情绪-游戏 |
| 感觉反应(S) | 感知觉 |
| 语言异常(L) | 沟通-语言表达 |

---

## 4. 生成规则

1. **任务目标 = 干预项目名称**：颗粒度细到具体项目（如"韵母a"而非"韵母训练"）
2. **短期目标 = 干预项目阶段名称**：按阶段顺序排列
3. **同一阶梯及领域下可有多个任务目标**
4. **不匹配时跳过**：评估薄弱领域在课包中找不到对应干预项目的，不生成 IEP 目标
5. **不能编造**：任何不在干预项目配置表中的项目名称或阶段名称都不能出现
6. **导出文件命名**：`{姓名}个别化教育计划.docx`

---

## 5. 优先级排序

```
P1：S级行为领域对应的干预项目
P2：得分率最低的功能发展领域对应的干预项目
P3：M级行为领域对应的干预项目
P4：其余薄弱领域
```
