# PEP-3 评估结果 → IEP 生成规则

```
=== INPUT ===

1. 个案基本信息
   - 姓名、性别、出生日期 → 计算生理年龄（月）
   - 诊断类型（档案中获取）：自闭症/发育迟缓/其他
   - 评估日期、评估师
   - IEP 时间范围：起止日期（如 2026-05-01 ~ 2026-07-31）→ 计算周期月数

2. PEP-3 评估数据（五部分）

   发展副测验（6个，共127题）：每题 2=P(通过) / 1=E(萌芽) / 0=F(不通过)
   ┌────────────────────┬────┬────┐
   │ 副测验              │代码│题数│
   │ 认知(语言/语前)      │CVP │ 34 │
   │ 语言表达             │EL  │ 25 │
   │ 语言理解             │RL  │ 19 │
   │ 小肌肉              │FM  │ 20 │
   │ 大肌肉              │GM  │ 15 │
   │ 模仿(视觉/动作)      │VMI │ 10 │
   └────────────────────┴────┴────┘

   行为副测验（4个，共45题）：每题 2=A(恰当) / 1=M(轻微) / 0=S(严重)
   ┌────────────────────┬────┬────┐
   │ 副测验              │代码│题数│
   │ 情感表达             │AE  │ 11 │
   │ 社交互动             │SR  │ 12 │
   │ 行为特征-非语言       │CMB │ 15 │
   │ 行为特征-语言         │CVB │ 11 │
   └────────────────────┴────┴────┘

   照顾者报告-儿童现时发展程度（6题）：0=<2岁 / 1=2-3岁 / 2=3-4岁 / 3=4-5岁 / 4=5-7岁
   → 照顾者视角的发展年龄估计，交叉验证测试员评估

   照顾者报告-诊断类别及程度（11类×2题）
   → 诊断类别：0=不是 / 1=不知道 / 2=是
   → 影响程度：0=严重 / 1=中度 / 2=轻微
   → 了解已有诊断，辅助优先级判断

   照顾者报告-问题行为/个人自理/适应行为（38题）
   ┌──────────┬────┬────────────────────────────┐
   │ 副测验    │代码│ 评分                        │
   │ 问题行为  │PB  │ 0=严重 / 1=轻微中度 / 2=没有 │
   │ 个人自理  │PSC │ 0=不能 / 1=部分 / 2=能做到   │
   │ 适应行为  │AB  │ 0=严重 / 1=轻微 / 2=恰当     │
   └──────────┴────┴────────────────────────────┘

3. 课包干预项目库
   - 指定课包（如 RICE1v1）中的全部干预项目
   - 每个项目含：name / territory / subdivide / age_bracket / stages[]

=== OUTPUT ===

输出：IEP 表格
┌──────────────┬──────────┬──────────┐
│ 阶梯及领域    │ 任务目标  │ 短期目标  │
│ {年龄段}-     │ {项目name}│ {阶段name}│
│ {territory}- │          │          │
│ {subdivide}  │          │          │
└──────────────┴──────────┴──────────┘

=== 推导规则 ===

一、计算各领域指标

   发展副测验：
   得分率 = P题数 / 总题数 × 100%（只计完全通过）
   发展年龄（月）= 得分率 × 生理年龄（月）
   薄弱判定：得分率 ≤ 65% → 薄弱

   行为副测验：
   A数 = 得2分题数，M数 = 得1分题数，S数 = 得0分题数
   行为评级：S数/总题数 ≥ 25% → S级，M数/总题数 ≥ 25% → M级，其余 → A级

   照顾者报告-问题行为(PB)：
   问题行为得分率 = 总得分 / 满分20 × 100%
   得分率 ≤ 50% → 问题行为严重，需优先干预

   照顾者报告-个人自理(PSC)：
   自理得分率 = 总得分 / 满分26 × 100%
   得分率 ≤ 50% → 补充生活自理目标

二、确定优先级排序（权重：诊断类型 > 问题行为 > 发展 > 行为）

   P0：诊断相关核心缺陷
       照顾者报告"自闭症=是 + 影响严重" → 社交情绪 + 沟通领域强制纳入
       照顾者报告"语言障碍=是 + 影响严重" → 沟通领域强制纳入
       照顾者报告"专注力不足/过度活跃症=是" → 学习行为领域强制纳入
       → 诊断类型决定哪些领域"必须有"，不受得分率门槛限制

   P1：照顾者报告-问题行为(PB)得分率 ≤ 50%
       → 家庭环境中的严重行为问题优先处理

   P2：发展副测验得分率 ≤ 35% 且 E项 ≥ 2
       → 最薄弱 + 有最近发展区 = 干预效果最好

   P3：发展副测验得分率 ≤ 35%

   P4：发展副测验得分率 36%~50% 且 E项 ≥ 2

   P5：发展副测验得分率 36%~50%

   P6：发展副测验得分率 51%~65%

   P7：行为副测验 S级

   P8：行为副测验 M级

   P9：照顾者报告-个人自理得分率 ≤ 50% → 补充生活自理目标

三、生理年龄与发展年龄差异 → 调整干预策略

   差距 = 生理年龄 - 发展年龄（月）

   差距 ≤ 6月：与同龄基本持平，常规干预
   差距 7~12月：轻度落后，增加该领域训练频次
   差距 13~24月：中度落后，该领域优先级提升一档
   差距 > 24月：显著落后，该领域强制纳入 IEP

   照顾者发展程度交叉验证：
   照顾者评估为 0（<2岁）的维度 → 对应领域优先级提升一档
   照顾者评估与测试员评估差距 > 1个年龄段 → 备注"评估差异，建议关注"

   全面落后（所有领域 ≤ 35%）：
   → 只选沟通 + 认知(模仿) + 社交情绪 3个基础领域
   → 其余下个周期再纳入

   能力不均衡（强弱差距 > 30%）：
   → 弱势领域正常纳入
   → 强势领域标注"可作为教学媒介"（利用强项带弱项）

四、限定干预目标数量（根据 IEP 时间范围梯度设计）

   ┌──────────┬──────┬───────┬──────────────────────────────┐
   │ 周期月数  │领域数 │总目标数│ 说明                          │
   │ 1个月    │ 1~2  │ 3~5   │ 短期集中突破，聚焦最核心问题    │
   │ 2个月    │ 2~3  │ 5~8   │ 核心领域为主，少量扩展          │
   │ 3个月    │ 3~5  │ 8~15  │ 标准周期，覆盖主要薄弱领域      │
   │ 6个月    │ 4~6  │ 12~20 │ 长周期，分阶段推进，可设中期检查 │
   │ 12个月   │ 5~8  │ 15~25 │ 年度计划，按季度分解子目标      │
   └──────────┴──────┴───────┴──────────────────────────────┘

   按发展年龄微调（在上表基础上）：
   < 2岁 → 领域数和目标数取下限
   2~3岁 → 取中间值
   3~5岁 → 取上限
   > 5岁 → 聚焦社交/学业/自理，领域数取中间值

   每领域任务目标：2~4个（上限5）
   每目标短期阶段：1~3个
   长周期（≥6个月）梯度设计：
   → 前1/3周期：基础目标（从萌芽技能切入）
   → 中1/3周期：进阶目标（巩固+扩展）
   → 后1/3周期：泛化目标（跨情境应用）

五、匹配干预项目

   领域映射（PEP-3 评估领域 → 课包领域）：
   ┌────────────────────┬───────────┬──────────────────┐
   │ PEP-3 领域          │ territory │ subdivide        │
   │ 小肌肉-FM           │ 粗大动作   │ 精细动作         │
   │ 大肌肉-GM           │ 粗大动作   │ 粗大动作         │
   │ 认知(语言/语前)-CVP  │ 认知      │ *               │
   │ 语言表达-EL          │ 沟通      │ 语言表达         │
   │ 语言理解-RL          │ 沟通      │ 语言理解         │
   │ 模仿(视觉/动作)-VMI  │ 认知      │ 模仿            │
   │ 社交互动-SR          │ 社交情绪   │ 社交基础、同伴互动│
   │ 情感表达-AE          │ 社交情绪   │ 情绪            │
   │ 行为特征-非语言-CMB   │ 学习行为   │ *               │
   │ 行为特征-语言-CVB     │ 沟通      │ 语言表达         │
   │ 问题行为-PB          │ 学习行为   │ *               │
   │ 个人自理-PSC         │ 生活自理   │ *               │
   │ 适应行为-AB          │ 社交情绪   │ *               │
   └────────────────────┴───────────┴──────────────────┘
   * = 匹配该 territory 下所有 subdivide

   年龄段匹配（按发展年龄，不按生理年龄）：
   0~23月 → 1-2岁 | 24~35月 → 2-3岁 | 36~47月 → 3-4岁
   48~59月 → 4-5岁 | ≥60月 → 5岁以上
   边界处理：优先选较低年龄段中未掌握的项目

   匹配排序：
   a. 与 E(萌芽) 技能直接对应的干预项目（最优先）
   b. 与 F(不通过) 技能对应的干预项目中，阶段最低的
   c. 该领域该年龄段的其他干预项目（补充）

   匹配失败 → 跳过，备注"XX领域建议补充专项干预方案"
   绝对不能编造不存在的项目名称

六、萌芽技能（E项）处理

   E项 = 最近发展区 = IEP 核心切入点
   → 干预起点 = E 说明中描述的当前能力水平
   → 短期目标 = 从 E 水平提升到 P 水平
   → E 说明中的"需要示范""需要辅助"等信息指导教学方法选择

=== 约束 ===

- 同一份评估数据 + 同一个课包 → 同一份 IEP
- 所有项目名称和阶段名称必须与课包完全一致
- 不能编造、改写或合并课包中的项目
- IEP 周期：由输入的时间范围决定，短期目标预期 2~4 周达成
- 导出文件：{姓名}个别化教育计划.docx
```
