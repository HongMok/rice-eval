# VB-MAPP 语言行为里程碑评估

## 概述

VB-MAPP（Verbal Behavior Milestones Assessment and Placement Program）是由 Mark Sundberg 博士开发的评估工具，基于 B.F. Skinner 的语言行为理论和应用行为分析（ABA）原理，用于评估孤独症及其他语言发育迟缓儿童的语言和社交技能。是目前 ABA 领域广泛使用的语言评估工具之一。

## 适用对象

适用发展水平 0 ~ 48 个月，主要面向孤独症谱系障碍、语言发育迟缓及其他发展障碍儿童，广泛应用于 ABA 机构的入学评估和教学计划制定。

## 评估结构

VB-MAPP 包含 5 大评估模块：

### 里程碑评估

核心模块，包含 170 个里程碑，分布在 3 个发展阶段（Level 1/2/3）、16 个领域。Level 1 对应 0-18 个月水平，涵盖提要求、命名、听者反应、视觉配对、游戏、社交、模仿、仿说、发音等基础领域；Level 2 新增 LRFFC、对话、集体技能、语言结构；Level 3 新增阅读、书写、数学。

### 障碍评估

评估 24 项常见的学习和语言习得障碍，涵盖行为问题、教学控制、语言缺陷、社交缺陷、学习障碍、动机问题、感觉问题等方面，每项 0-4 分，分数越高障碍越严重。

### 转衔评估

评估儿童从密集教学环境向限制较少的教育环境过渡的准备程度，包含 18 个评估领域，每项 0-5 分。

### 任务分析和技能追踪

对里程碑评估中的每个领域进行更细致的技能分解，用于制定具体的教学目标。

### 安置和IEP目标

基于评估结果，提供教学安置建议和 IEP 目标建议。

## 评分体系

里程碑评分：达标 1 分、部分达标 0.5 分、未达标 0 分。总分 0-45 分为第一阶段（0-18个月水平），46-90 分为第二阶段（18-30个月水平），91-170 分为第三阶段（30-48个月水平）。

## 评估特点

- 基于 ABA 原理，评估结果可直接指导教学，与 ABA 教学策略无缝衔接
- 里程碑式评估，清晰展示儿童当前发展水平和下一步目标
- 里程碑 + 障碍 + 转衔多模块评估，全面覆盖儿童的能力和需求
- 基于 Skinner 的语言行为分类（Mand/Tact/Echoic/Intraverbal 等），精准定位语言技能
- 里程碑评估约 45-90 分钟，完整评估（含障碍和转衔）约 2-3 小时
