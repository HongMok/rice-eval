# VB-MAPP 语言行为里程碑评估

## 概述

VB-MAPP（Verbal Behavior Milestones Assessment and Placement Program）是由 Mark Sundberg 博士开发的评估工具，基于 B.F. Skinner 的语言行为理论和应用行为分析（ABA）原理，用于评估孤独症及其他语言发育迟缓儿童的语言和社交技能。VB-MAPP 是目前 ABA 领域最广泛使用的语言评估工具之一。

## 适用对象

- **适用年龄**：0岁 ~ 48个月（发展水平）
- **适用人群**：孤独症谱系障碍、语言发育迟缓及其他发展障碍儿童
- 广泛应用于 ABA 机构的入学评估和教学计划制定

## 评估结构

VB-MAPP 包含 **5大评估模块**：

### 1. 里程碑评估（Milestones Assessment）

里程碑评估是 VB-MAPP 的核心模块，包含 **170个里程碑**，分布在 **3个发展阶段**、**16个领域**：

#### Level 1（0-18个月，每领域5个里程碑）

| 领域 | 英文 | 评估内容 |
|------|------|----------|
| 提要求 | Mand | 用语言表达需求和愿望 |
| 命名 | Tact | 看到物品/事件能说出名称 |
| 听者反应 | Listener | 理解并执行他人指令 |
| 视觉配对 | VP-MTS | 视觉辨别和匹配能力 |
| 游戏 | Play | 独立游戏和休闲技能 |
| 社交 | Social | 社交互动和回应 |
| 模仿 | Imitation | 动作和口型模仿 |
| 仿说 | Echoic | 语音模仿能力 |
| 发音 | Vocal | 自发发音的清晰度和多样性 |

#### Level 2（18-30个月，新增领域）

在 Level 1 基础上新增：
- **LRFFC**（听者功能/特征/类别反应）
- **对话**（Intraverbal）
- **集体技能**（Group）
- **语言结构**（Linguistics）

#### Level 3（30-48个月，新增领域）

在 Level 2 基础上新增：
- **阅读**（Reading）
- **书写**（Writing）
- **数学**（Math）

### 2. 障碍评估（Barriers Assessment）

评估 **24项** 常见的学习和语言习得障碍，每项 0-4 分（越高障碍越严重）：

| 障碍类别 | 示例 |
|----------|------|
| 行为问题 | 自伤、攻击、逃避等 |
| 教学控制 | 配合度、注意力等 |
| 语言缺陷 | 命名缺陷、仿说缺陷、对话缺陷等 |
| 社交缺陷 | 社会技能缺陷、眼神接触等 |
| 学习障碍 | 依赖辅助、不能泛化、猜想式回答等 |
| 动机问题 | 薄弱动机、依赖强化物等 |
| 感觉问题 | 自我刺激、感觉防御等 |

### 3. 转衔评估（Transition Assessment）

评估儿童从密集教学环境向限制较少的教育环境过渡的准备程度，包含 **18个评估领域**，每项 0-5 分，分为三大类别：

| 类别 | 评估领域 |
|------|----------|
| 第一类：VB-MAPP得分和独立学习能力 | 里程碑得分、障碍得分、负面行为、教室常规、集体技能、社交技能、独立学业工作 |
| 第二类：学习模式 | 泛化、强化物泛化、技能获得速度、技能维持、自然环境学习、未经训练的转换 |
| 第三类：自理、自发和自主 | 适应性、自发行为、自我导向休闲、自理能力、入厕技能、进餐技能 |

### 4. 任务分析和技能追踪（Task Analysis）

对里程碑评估中的每个领域进行更细致的技能分解，用于制定具体的教学目标。

### 5. 安置和IEP目标（Placement and IEP Goals）

基于评估结果，提供教学安置建议和 IEP 目标建议。

## 评分体系

### 里程碑评分

| 评分 | 分值 | 含义 |
|------|------|------|
| 达标 | 1分 | 完全达到该里程碑标准 |
| 部分达标 | 0.5分 | 部分达到，技能不稳定 |
| 未达标 | 0分 | 未达到该里程碑标准 |

### 发展阶段判定

| 里程碑总分 | 发展阶段 |
|-----------|----------|
| 0-45分 | 第一阶段（0-18个月水平） |
| 46-90分 | 第二阶段（18-30个月水平） |
| 91-170分 | 第三阶段（30-48个月水平） |

## 评估结果

### 里程碑计分总表
- 网格图展示各领域各阶段的得分情况
- 填色规则：1分=全填色，0.5分=半填色，0分=空白
- 直观展示儿童在各领域的发展水平

### 障碍评估报告
- 24项障碍的得分和严重度评级
- 重点关注得分 ≥3 的障碍项（需优先干预）

### 转衔评估报告
- 三类转衔技能的得分和准备度评级
- 总分对应的安置建议（个训为主/小组为辅/融合教育）

## 评估特点

1. **基于 ABA 原理**：评估结果可直接指导教学，与 ABA 教学策略无缝衔接
2. **里程碑式评估**：清晰展示儿童当前发展水平和下一步目标
3. **多模块评估**：里程碑 + 障碍 + 转衔，全面评估儿童的能力和需求
4. **语言行为分类**：基于 Skinner 的语言行为分类（Mand/Tact/Echoic/Intraverbal 等），精准定位语言技能
5. **施测时间**：里程碑评估约 45-90分钟，完整评估（含障碍和转衔）约 2-3小时

## 核心术语

| 术语 | 英文 | 含义 |
|------|------|------|
| 提要求 | Mand | 由动机驱动的语言行为，用于获取想要的物品或活动 |
| 命名 | Tact | 由非语言刺激（看到、听到、闻到等）控制的语言行为 |
| 仿说 | Echoic | 重复他人语言的行为 |
| 对话 | Intraverbal | 由他人语言控制的语言行为（如回答问题） |
| 听者反应 | Listener | 对他人语言做出非语言反应（如执行指令） |
| LRFFC | Listener Responding by Function, Feature, and Class | 根据功能、特征和类别做出听者反应 |

## 注意事项

- 评估师需接受 VB-MAPP 专项培训
- 评估前需准备充足的强化物和教学材料
- 里程碑评估建议在自然情境中进行，避免过度结构化
- 障碍评估需结合日常观察，不能仅依赖单次评估
- 评估结果应定期更新（建议每 3-6 个月重新评估）
- 评估结果用于教学计划制定，不能替代医学诊断
