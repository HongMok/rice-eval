# C-PEP 评估结果 → IEP 生成规则

> 本文档定义 C-PEP 评估结果如何转化为个别化教育计划（IEP）。
> 核心原则：IEP 中的所有内容必须来自课包干预项目配置表，不能自行编造。

---

## 1. IEP 表格结构

同 PEP-3，三列：阶梯及领域、任务目标、短期目标。

---

## 2. 薄弱领域筛选

### 2.1 阈值

```
薄弱领域 = 得分率 ≤ 65% 的领域
```

### 2.2 排序

按得分率升序排列。

### 2.3 萌芽技能优先

C-PEP 特有：`中间反应`（partial）数量高的领域代表最近发展区，优先从这些领域的干预项目入手。

---

## 3. 领域映射

C-PEP 评估领域 → 课包干预领域：

| C-PEP 领域 | 课包领域 |
|------------|---------|
| 感知觉 | 感知觉 |
| 精细动作 | 精细动作 |
| 粗大动作 | 粗大动作 |
| 模仿 | 模仿 |
| 认知 | 认知 |
| 语言理解 | 沟通-语言理解 |
| 语言表达 | 沟通-语言表达 |

---

## 4. 生成规则

1. **任务目标 = 干预项目名称**，颗粒度细到具体项目
2. **短期目标 = 干预项目阶段名称**，按阶段顺序排列
3. **不匹配时跳过**，**不能编造**
4. **导出文件命名**：`{姓名}个别化教育计划.docx`

---

## 5. 优先级排序

```
P1：得分率 ≤ 45% 且 partial 占比 ≥ 15% 的领域（最薄弱+有萌芽，优先干预）
P2：得分率 ≤ 45% 的领域（最薄弱）
P3：得分率 46%~55% 的领域
P4：得分率 56%~65% 的领域
```
