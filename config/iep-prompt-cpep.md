# C-PEP-3 评估结果 → IEP 生成规则

> 本文档定义 C-PEP-3 评估结果如何转化为个别化教育计划（IEP）。
> 核心原则：IEP 中的所有干预项目必须来自指定课包的干预项目配置表，不能自行编造。

---

## 1. 评估数据结构

C-PEP-3 评估数据包含三部分：

| 板块 | CSV一级分类 | 行数 | 评分体系 | IEP 用途 |
|------|-----------|------|---------|---------|
| 发展量表 | 发展 | 96 | 2=P(通过), 1=E(萌芽), 0=F(不通过) | E项 = 最近发展区，优先干预 |
| 病理行为量表 | 病理学 | 44 | 2=A(恰当), 1=M(轻微), 0=S(严重) | S/M项 = 行为干预目标 |
| 补充项目 | 补充项目 | 32 | 1=H(通过), 0=L(不通过) | 精准定位能力上下限，调整干预起点 |

## 3. IEP 编写流程

```
Step 1：汇总评估数据
  输入：C-PEP-3 答题数据 + 儿童基本信息（姓名、出生日期、诊断）
  输出：各领域得分率、E(萌芽)项清单、S(严重)项清单、补充项目H/L结果

Step 2：确定发展年龄
  输入：各领域得分率 + 儿童生理年龄
  输出：各领域发展年龄（月）
  规则：发展年龄（月）= 得分率 × 生理年龄（月）
  注意：补充项目不参与发展年龄计算

Step 3：筛选薄弱领域
  输入：各领域得分率 + 病理行为评级
  输出：需要干预的领域列表（按优先级排序）
  规则：见第 5 节

Step 4：确定干预年龄段
  输入：各领域发展年龄
  输出：每个领域对应的课包年龄段
  规则：见第 9 节

Step 5：匹配干预项目
  输入：薄弱领域 + 干预年龄段 + 课包 + E项清单 + 补充项目结果
  输出：匹配到的干预项目列表
  规则：见第 10 节

Step 6：控制数量 + 排序
  输入：匹配到的干预项目
  输出：精简后的 IEP 条目
  规则：见第 8 节

Step 7：生成 IEP 表格
  输出：阶梯及领域 / 任务目标 / 短期目标
```

---

## 4. IEP 表格结构

| 列 | 数据来源 | 示例 |
|----|---------|------|
| 阶梯及领域 | `{年龄段}-{territory}-{subdivide}` | `2-3岁-认知-模仿` |
| 任务目标 | 干预项目 `name` | `动作模仿` |
| 短期目标 | 干预项目阶段 `stage.name` | `模仿单一动作` |

---

## 5. 薄弱领域筛选

### 5.1 发展量表（9个领域）

```
领域得分 = P×2 + E×1 + F×0
领域满分 = 总题数 × 2
得分率 = 领域得分 / 领域满分 × 100%
薄弱领域 = 得分率 ≤ 65%
```

### 5.2 病理行为量表（5个领域）

```
领域严重率 = S数 / 总题数 × 100%
纳入 IEP = 严重率 ≥ 20%（即评级为 S 或 M）
```

---

## 6. 优先级排序

```
P0：病理行为量表中严重率 ≥ 50% 的领域（S级，严重行为问题）
    → 特别关注：感觉异常、情感失调、人际关系障碍
    → 这些问题不解决，技能训练效果会大打折扣

P1：发展量表得分率 ≤ 35% 且 E(萌芽)项 ≥ 2 个的领域
    → 最薄弱 + 有最近发展区 = 干预效果最好

P2：发展量表得分率 ≤ 35% 的领域

P3：发展量表得分率 36%~50% 且 E(萌芽)项 ≥ 2 个的领域

P4：发展量表得分率 36%~50% 的领域

P5：发展量表得分率 51%~65% 的领域

P6：病理行为量表中严重率 20%~49% 的领域（M级）
```

| C-PEP-3 二级分类 | 课包 territory | 课包 subdivide | 说明 |
|-----------------|---------------|---------------|------|
| 精细动作 | 粗大动作 | 精细动作 | |
| 粗大动作 | 粗大动作 | 粗大动作 | |
| 模仿(动作) | 认知 | 模仿 | |
| 模仿(口语) | 发音 | * | 口语模仿对应发音训练 |
| 知觉(视觉) | 认知 | 视觉 | |
| 知觉(听觉) | 认知 | 听觉 | |
| 认知表现 | 认知 | * | 匹配认知下所有subdivide |
| 口语认知 | 沟通 | 语言表达 | |
| 手眼协调 | 粗大动作 | 精细动作 | |
| 感觉 | 认知 | 听觉、视觉 | 感觉异常通过感统训练 |
| 情感 | 社交情绪 | 情绪 | |
| 人际关系、合作行为及对人的兴趣 | 社交情绪 | 社交基础、同伴互动 | |
| 游戏及对物的兴趣 | 游戏 | * | |
| 语言 | 沟通 | 语言表达 | |

## 8. IEP 项目数量控制

### 8.1 领域和目标数量

| 项目 | 推荐 | 上限 |
|------|------|------|
| 干预领域数 | 3~5 个 | 6 个 |
| 每领域任务目标 | 2~4 个 | 5 个 |
| 每目标短期阶段 | 1~3 个 | 全部阶段 |
| 整份 IEP 总目标 | 8~15 个 | 20 个 |

原则：宁可少而精，不可多而散。每个领域需要足够训练时间才能见效。

### 8.2 按发展年龄调整

| 发展年龄 | 领域数 | 总目标数 | 说明 |
|---------|--------|---------|------|
| < 2岁 | 2~3 | 6~8 | 低龄注意力短，目标少而集中 |
| 2~3岁 | 3~4 | 8~12 | 逐步增加覆盖 |
| 3~5岁 | 4~5 | 10~15 | 学龄前关键期 |
| > 5岁 | 3~5 | 8~12 | 聚焦社交、学业准备、生活自理 |

### 8.3 特殊情况

**全面落后（所有领域 ≤ 35%）**：优先选沟通 + 认知(模仿) + 社交情绪 3个基础领域，其余下个周期再纳入。

**能力不均衡（强弱差距 > 30%）**：弱势领域正常纳入，强势领域标注"可作为教学媒介"。

---

## 9. 年龄段匹配

**按发展年龄选课包年龄段，不按生理年龄。**

| 领域发展年龄（月） | 课包年龄段 |
|-----------------|---------|
| 0~23 | 1-2岁 |
| 24~35 | 2-3岁 |
| 36~47 | 3-4岁 |
| 48~59 | 4-5岁 |
| ≥ 60 | 5岁以上 |

边界处理：发展年龄在边界附近时，同时查看相邻两个年龄段，优先选较低年龄段中未掌握的项目。

---

## 10. 干预项目匹配规则

### 10.1 匹配流程

```
对每个薄弱领域：
  1. 通过领域映射表（第7节）→ 得到课包 territory + subdivide
  2. 筛选：该 territory + subdivide + 干预年龄段 的所有干预项目
  3. 排序匹配：
     a. 与 E(萌芽) 技能直接对应的干预项目（最优先）
     b. 与 F(不通过) 技能对应的干预项目中，阶段最低的
     c. 该领域该年龄段的其他干预项目（补充）
  4. 结合补充项目结果调整起点（见第11节）
  5. 取前 2~4 个作为该领域的任务目标
```

### 10.2 阶段选择

```
每个干预项目的阶段按编号顺序排列：
  → 短期目标 = 从阶段 1 开始列出
  → 如果评估显示儿童已具备某阶段能力，可跳过
  → 每个任务目标列 1~3 个阶段
```

### 10.3 匹配失败

```
课包中找不到对应干预项目时：
  → 跳过，不生成 IEP 条目
  → 在备注中标注"XX领域建议补充专项干预方案"
  → 绝对不能编造不存在的项目名称
```

## 11. 补充项目对 IEP 的影响

补充项目**不直接生成 IEP 条目**，而是用于**精准调整干预起点和难度**。

### 11.1 调整规则

| 主项目结果 | 补充项目 | 补充结果 | IEP 调整 |
|-----------|---------|---------|---------|
| P(通过) | (H)高阶 | H(通过) | 该技能跳过，不纳入 IEP |
| P(通过) | (H)高阶 | L(不通过) | 将高阶技能作为 IEP 目标 |
| F(不通过) | (L)低阶 | H(通过) | 干预从中间阶段开始，不从零起步 |
| F(不通过) | (L)低阶 | L(不通过) | 干预从最基础的前备技能开始 |
| E(萌芽) | (L)低阶 | H(通过) | 最佳干预切入点，优先纳入 IEP |

### 11.2 示例

```
主项目 #59（用手势表达5种物品的用途）= E(萌芽)
补充项目 #121（59(L)：用手势表达实物的用途）= H(通过)
→ IEP 目标：从实物过渡到图片的手势表达（中间阶段开始）
```

---

## 12. 萌芽技能处理

E(萌芽)项是 IEP 的核心切入点——代表儿童"快要会但还不稳定"的能力（最近发展区），干预从这里入手成功率最高。

C-PEP-3 每个项目都有详细的 P/E/F 评分说明，E 的说明可直接用于：
1. **确定干预起点**：E 说明 = 儿童当前能力水平
2. **设定短期目标**：从 E 水平 → P 水平
3. **选择教学策略**：E 说明中的"需要示范""需要辅助"等信息指导教学方法

---

## 13. IEP 周期

```
IEP 周期：3 个月
短期目标预期：每个目标 2~4 周达成
评估频率：每月评估一次短期目标达成情况
```

---

## 14. 输出格式

```
| 阶梯及领域 | 任务目标 | 短期目标 |
|-----------|---------|---------|
| {年龄段}-{territory}-{subdivide} | {project.name} | {stage.name} |
```

导出文件：`{姓名}个别化教育计划.docx`

---

## 15. 确定性要求

- 同一份评估数据 + 同一个课包 → 同一份 IEP
- 所有项目名称和阶段名称必须与课包完全一致
- 不能编造、改写或合并课包中的项目
