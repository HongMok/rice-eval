# C-PEP-3 评估结果 → IEP 生成规则

```
=== INPUT ===

1. 个案基本信息
   - 姓名、性别、出生日期 → 计算生理年龄（月）
   - 诊断类型（档案中获取）：自闭症/发育迟缓/其他
   - 评估日期、评估师
   - IEP 时间范围：起止日期（如 2026-05-01 ~ 2026-07-31）→ 计算周期月数

2. C-PEP-3 评估数据（三部分）

   发展量表（96题）：9个领域，每题 2=P(通过) / 1=E(萌芽) / 0=F(不通过)
   ┌──────────┬────┐
   │ 领域      │题数│
   │ 模仿(动作) │ 10 │
   │ 模仿(口语) │  4 │
   │ 知觉(视觉) │ 11 │
   │ 知觉(听觉) │  4 │
   │ 精细动作   │ 10 │
   │ 粗大动作   │ 11 │
   │ 手眼协调   │ 14 │
   │ 认知表现   │ 20 │
   │ 口语认知   │ 12 │
   └──────────┴────┘

   病理行为量表（44题）：5个领域，每题 2=A(恰当) / 1=M(轻微) / 0=S(严重)
   ┌──────────────────────────┬────┐
   │ 领域                      │题数│
   │ 感觉                      │ 12 │
   │ 情感                      │  8 │
   │ 人际关系、合作行为及对人的兴趣│  8 │
   │ 游戏及对物的兴趣            │  6 │
   │ 语言                      │ 10 │
   └──────────────────────────┴────┘

   补充项目（32题）：每题 1=H(通过) / 0=L(不通过)
   → 不参与得分率计算，仅用于调整干预起点

3. 课包干预项目库
   - 指定课包（如 RICE1v1）中的全部干预项目
   - 每个项目含：name / territory / subdivide / age_bracket / stages[]

=== OUTPUT ===

输出：IEP 表格
┌──────────────┬──────────┬──────────┐
│ 阶梯及领域    │ 任务目标  │ 短期目标  │
│ {年龄段}-     │ {项目name}│ {阶段name}│
│ {territory}- │          │          │
│ {subdivide}  │          │          │
└──────────────┴──────────┴──────────┘

=== 推导规则 ===

一、计算各领域指标

   得分率 = P题数 / 总题数 × 100%（只计完全通过）
   发展年龄（月）= 得分率 × 生理年龄（月）
   病理严重率 = S题数 / 总题数 × 100%

   发展量表薄弱判定：得分率 ≤ 65% → 薄弱
   病理行为评级：严重率 ≥ 50% → S级，20%~49% → M级，< 20% → A级

二、确定优先级排序（权重：诊断类型 > 病理行为 > 发展）

   P0：诊断相关核心缺陷
       自闭症诊断 → 人际关系 + 口语认知 领域强制纳入（即使得分率 > 65%）
       语言发育迟缓 → 口语认知 领域强制纳入
       → 诊断类型决定哪些领域"必须有"，不受得分率门槛限制

   P1：病理行为 S级（严重率 ≥ 50%）
       → 严重行为问题不解决，技能训练效果大打折扣
       → 特别关注：感觉异常、情感失调、人际关系障碍

   P2：病理行为 M级（严重率 20%~49%）
       → 需在教学中持续关注

   P3：发展量表得分率 ≤ 35% 且 E项 ≥ 2
       → 最薄弱 + 有最近发展区 = 干预效果最好

   P4：发展量表得分率 ≤ 35%
       → 最薄弱领域

   P5：发展量表得分率 36%~50% 且 E项 ≥ 2

   P6：发展量表得分率 36%~50%

   P7：发展量表得分率 51%~65%

三、生理年龄与发展年龄差异 → 调整干预策略

   差距 = 生理年龄 - 发展年龄（月）

   差距 ≤ 6月：与同龄基本持平，常规干预
   差距 7~12月：轻度落后，增加该领域训练频次
   差距 13~24月：中度落后，该领域优先级提升一档
   差距 > 24月：显著落后，该领域强制纳入 IEP

   全面落后（所有领域 ≤ 35%）：
   → 只选口语认知 + 模仿 + 人际关系 3个基础领域
   → 其余下个周期再纳入

   能力不均衡（强弱差距 > 30%）：
   → 弱势领域正常纳入
   → 强势领域标注"可作为教学媒介"（利用强项带弱项）

四、限定干预目标数量（根据 IEP 时间范围梯度设计）

   ┌──────────┬──────┬───────┬──────────────────────────────┐
   │ 周期月数  │领域数 │总目标数│ 说明                          │
   │ 1个月    │ 1~2  │ 3~5   │ 短期集中突破，聚焦最核心问题    │
   │ 2个月    │ 2~3  │ 5~8   │ 核心领域为主，少量扩展          │
   │ 3个月    │ 3~5  │ 8~15  │ 标准周期，覆盖主要薄弱领域      │
   │ 6个月    │ 4~6  │ 12~20 │ 长周期，分阶段推进，可设中期检查 │
   │ 12个月   │ 5~8  │ 15~25 │ 年度计划，按季度分解子目标      │
   └──────────┴──────┴───────┴──────────────────────────────┘

   按发展年龄微调（在上表基础上）：
   < 2岁 → 领域数和目标数取下限
   2~3岁 → 取中间值
   3~5岁 → 取上限
   > 5岁 → 聚焦社交/学业/自理，领域数取中间值

   每领域任务目标：2~4个（上限5）
   每目标短期阶段：1~3个
   长周期（≥6个月）梯度设计：
   → 前1/3周期：基础目标（从萌芽技能切入）
   → 中1/3周期：进阶目标（巩固+扩展）
   → 后1/3周期：泛化目标（跨情境应用）

五、匹配干预项目

   领域映射规则（评估领域 → 课包领域）：
   不写死映射表，按以下规则动态匹配课包中的干预项目：
   1. 语义匹配：评估领域名称与课包 territory/subdivide 进行语义相似度匹配
      - 优先精确匹配（如"粗大动作"匹配课包中含"粗大动作"的项目）
      - 次选模糊匹配（如"手眼协调"匹配课包中含"精细动作"关键词的项目）
   2. 能力维度匹配：根据评估领域考察的核心能力维度，匹配课包中训练相同能力的项目
      - 语言类领域（口语认知/语言）→ 匹配课包中语言表达/语言理解相关项目
      - 认知类领域（认知表现/知觉）→ 匹配课包中认知相关项目
      - 运动类领域（精细动作/手眼协调/粗大动作）→ 匹配课包中精细动作/粗大动作相关项目
      - 模仿类领域（模仿-动作/模仿-口语）→ 匹配课包中模仿/发音相关项目
      - 社交类领域（情感/人际关系）→ 匹配课包中社交/情绪相关项目
      - 感觉类领域（感觉）→ 匹配课包中感觉/认知相关项目
      - 游戏类领域（游戏及对物的兴趣）→ 匹配课包中游戏相关项目
   3. 年龄段过滤：匹配结果按干预年龄段筛选
   4. 一个评估领域可匹配课包中多个 territory/subdivide

   年龄段匹配（按发展年龄，不按生理年龄）：
   0~23月 → 1-2岁 | 24~35月 → 2-3岁 | 36~47月 → 3-4岁
   48~59月 → 4-5岁 | ≥60月 → 5岁以上
   边界处理：优先选较低年龄段中未掌握的项目

   匹配排序：
   a. 与 E(萌芽) 技能直接对应的干预项目（最优先）
   b. 与 F(不通过) 技能对应的干预项目中，阶段最低的
   c. 该领域该年龄段的其他干预项目（补充）

   补充项目调整起点：
   主项目=E + 补充(L)=H → 最佳切入点，从中间阶段开始
   主项目=F + 补充(L)=H → 干预从中间阶段开始，不从零起步
   主项目=F + 补充(L)=L → 干预从最基础的前备技能开始
   主项目=P + 补充(H)=L → 将高阶技能作为 IEP 目标

   匹配失败 → 跳过，备注"XX领域建议补充专项干预方案"
   绝对不能编造不存在的项目名称

六、萌芽技能（E项）处理

   E项 = 最近发展区 = IEP 核心切入点
   → 干预起点 = E 说明中描述的当前能力水平
   → 短期目标 = 从 E 水平提升到 P 水平
   → E 说明中的"需要示范""需要辅助"等信息指导教学方法选择

=== 约束 ===

- 同一份评估数据 + 同一个课包 → 同一份 IEP
- 所有项目名称和阶段名称必须与课包完全一致
- 不能编造、改写或合并课包中的项目
- IEP 周期：由输入的时间范围决定，短期目标预期 2~4 周达成
- 导出文件：{姓名}个别化教育计划.docx
```
