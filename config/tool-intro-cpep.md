# C-PEP 孤独症儿童发展评估

## 概述

C-PEP（Chinese Psychoeducational Profile）是中国残疾人联合会组织编制的孤独症儿童发展评估工具，在 PEP 系列量表基础上，结合中国儿童发展特点进行本土化改编。该量表广泛应用于全国残联定点康复机构的入学评估、阶段性评估和康复效果评价。

## 适用对象

- **适用年龄**：0岁 ~ 6岁
- **适用人群**：孤独症谱系障碍儿童及其他发育迟缓儿童
- 广泛应用于残联定点康复机构的入学评估和阶段性评估

## 评估结构

C-PEP 涵盖 **7大功能发展领域** 和 **5个行为病理领域**：

### 功能发展领域（共285题）

| 领域 | 题数 | 评估内容 |
|------|------|----------|
| 感知觉 | 55 | 视觉、听觉、触觉、嗅觉、味觉、本体觉、前庭觉 |
| 精细动作 | 36 | 手指精细动作、手眼协调、双手协调、工具使用 |
| 粗大动作 | 72 | 坐、站、走、跑跳、平衡、球类 |
| 模仿 | 14 | 动作模仿、口型模仿 |
| 认知 | 48 | 注意力、记忆、概念、推理 |
| 语言理解 | 30 | 指令理解、语义理解 |
| 语言表达 | 30 | 口语表达、沟通意图 |

### 行为病理领域

| 领域 | 评估内容 |
|------|----------|
| 情感 | 情绪表达和调节能力 |
| 语言 | 语言使用异常（仿说、代词混淆等） |
| 人际关系 | 社交互动、合作行为及对人的兴趣 |
| 感觉模式 | 感觉反应异常（高敏/低敏） |
| 游戏及物品喜好 | 游戏方式和兴趣范围 |

## 评分体系

### 功能发展评分

| 评分 | 分值 | 含义 |
|------|------|------|
| 通过（P） | 1分 | 独立完成，表现符合预期 |
| 中间反应（E） | 0.5分 | 部分完成或需要提示，技能处于萌芽阶段 |
| 不通过（F） | 0分 | 无反应或无法完成 |

### 行为病理评分

| 评分 | 含义 |
|------|------|
| A（适当） | 行为表现适当，无明显异常 |
| M（轻度） | 轻度异常，需关注 |
| S（严重） | 严重异常，需优先干预 |

## 评估结果

### C-PEP-3 记分册
- **功能发展汇总表**：7个领域的 P/E/F 分数统计
- **行为领域汇总表**：5个行为领域的 A/M/S 统计
- **发展评分**：所有领域 P 分总和

### 发展功能量表侧面图
- X轴：7个功能领域
- Y轴：年龄当量（0~6岁）
- 实线（P分）：反映已掌握的技能水平
- 虚线（P+E分）：反映潜在发展水平（含萌芽技能）
- 两线之间的区域即为"最近发展区"

### 行为病理行为靶心图
- 5个行为领域均匀分布在圆周上
- 从圆心向外：S级涂黑 → M级涂灰 → A级留白
- 直观展示行为问题的严重程度和分布

## 评估特点

1. **本土化设计**：结合中国儿童发展特点，更贴合国内康复实践
2. **残联体系对接**：评估结果可直接对接残联康复服务体系
3. **萌芽技能识别**：中间反应（E）分数反映儿童的最近发展区，为干预提供切入点
4. **多维度评估**：功能发展 + 行为病理双维度，评估更全面
5. **施测时间**：约 40-60分钟

## 注意事项

- 评估前需确保与儿童建立良好配合关系
- 评估环境应安静、减少干扰
- 评估师需经过专业培训
- 中间反应（E）项目数量反映萌芽技能，在制定 IEP 时应重点关注
- 评估结果应结合日常观察和家长反馈综合判断
