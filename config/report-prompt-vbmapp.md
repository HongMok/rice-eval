# VB-MAPP 评估报告 — 计算规则与撰写模板

> 本文档定义 VB-MAPP 评估报告的所有计算规则和撰写模板。
> 同一份评估数据，必须产出完全相同的报告结果。

---

## 报告整体结构

```
基本信息（姓名/性别/出生日期/测评日期）

第一部分：里程碑评估
  ├─ 计分总表（网格图）
  ├─ 表后总结（整体情况+优势+薄弱+小结，一段完整文本）
  └─ 16个领域逐项分析

第二部分：障碍评估
  ├─ 计分总表
  ├─ 表后总结（整体情况+高分障碍+对教学的影响+小结）
  └─ 24项障碍逐项分析

第三部分：转衔评估
  ├─ 计分总表
  ├─ 表后总结（整体准备度+优势+需支持领域+安置建议+小结）
  └─ 18项转衔领域逐项分析
```

## 2. 里程碑评估 — 计算规则

### 2.1 评分体系

1分（达标）、0.5分（部分达标）、0分（未达标）

### 2.2 三级阶段

| 阶段 | 月龄 | 满分 |
|------|------|------|
| Level 1 | 0-18月 | 45 |
| Level 2 | 18-30月 | 60 |
| Level 3 | 30-48月 | 65 |

### 2.3 16个领域满分表

| 领域 | L1 | L2 | L3 | 满分 |
|------|----|----|----|----|
| 提要求(Mand) | 5 | 5 | 5 | 15 |
| 命名(Tact) | 5 | 5 | 5 | 15 |
| 听者反应(LR) | 5 | 5 | 5 | 15 |
| 视觉配对(VP-MTS) | 5 | 5 | 5 | 15 |
| 游戏(Play) | 5 | 5 | - | 10 |
| 社交(Social) | 5 | 5 | 5 | 15 |
| 模仿(Imitation) | 5 | 5 | - | 10 |
| 仿说(Echoic) | 5 | 5 | - | 10 |
| 发音(SVB) | 5 | - | - | 5 |
| LRFFC | - | 5 | 5 | 10 |
| 对话(IV) | - | 5 | 5 | 10 |
| 集体技能(Group) | - | 5 | 5 | 10 |
| 语言结构(Ling) | - | 5 | 5 | 10 |
| 数学(Math) | - | - | 5 | 5 |
| 阅读(Reading) | - | - | 5 | 5 |
| 书写(Writing) | - | - | 5 | 5 |
| **合计** | **45** | **60** | **65** | **170** |

### 2.4 得分计算

```
领域得分 = 该领域在所有出现的Level中的得分之和
领域得分率 = 领域得分 / 领域满分 × 100%
里程碑总分 = 所有领域得分之和（满分170）
```

### 2.5 发展阶段判定

| 总分 | 阶段 |
|------|------|
| 0~45 | Level 1（0-18月） |
| 46~90 | Level 2（18-30月） |
| 91~170 | Level 3（30-48月） |

### 2.6 领域评级

| 得分率 | 评级 |
|--------|------|
| ≥ 70% | 表现良好 |
| 40%~69% | 发展中 |
| < 40% | 需重点关注 |

## 3. 里程碑评估 — 报告撰写

### 3.1 报告定位

里程碑评估报告的重点是**结果解读和干预方向说明**，而非分数呈现。家长关心的是：孩子当前存在哪些核心能力不足、为什么要设立这些短期目标、这些目标将如何帮助孩子改善当前的问题、后续干预会如何具体落实。

### 3.2 计分总表

以网格图形式呈现所有领域在三个等级的得分。（此部分为数据可视化，不需要AI生成文字）

### 3.3 表后总结

整体情况、优势领域、薄弱领域、评估小结融合为一段连续的专业文本：

```
儿童当前实际年龄为X岁X个月，VB-MAPP 里程碑总分为 __/170，
技能主要集中在 Level __（大致对应__至__个月发展水平）。
其已具备______等基础能力，但在______等领域仍需系统教学。

从各领域得分看，相对优势领域为______（得分率___%）和______（得分率___%），
这些优势是可以利用的行为库，教学设计应利用优势作为切入点支持薄弱领域。

当前明显薄弱的领域为______（得分率___%）和______（得分率___%），
需要区分技能缺陷和表现缺陷，优先安排系统教学。

综合来看，各领域发展并不均衡，尤其是______与______之间的差距提示______。
后续教学应结合领域间关系进行课程设计，
同时需要结合障碍评估结果分析影响技能发展的障碍因素。
```

### 3.4 各领域逐项分析（16个领域）

每个领域的分析必须以**连续、专业、可读的段落**方式呈现，禁止使用"一、二、三"或"1、2、3"的列表形式。读者应能清楚看到：该领域是什么、孩子现在会什么、不会什么、为什么重要，以及接下来应该怎么教。

**内在结构（不在报告中显式标注，但每段必须覆盖）**：

1. **领域定义与评估目的**：用 Skinner 语言行为理论术语定义该领域，说明评估该能力的意义
2. **得分与发展水平**：结合具体得分、总分和所处里程碑阶段，说明当前发展水平
3. **基于具体条目的表现分析**：
   - 已得分条目 → 描述儿童目前已经能够完成的技能（具体到可观察行为）
   - 未得分/部分得分条目 → 描述尚未稳定掌握的主要困难
   - 不能只写"该领域得分较低"，必须说明会什么、不会什么、差距在哪里
4. **教学意义**：解释这些结果对后续教学计划的意义
5. **干预建议**：提出与未掌握条目和当前发展阶段相对应的具体教学建议

**示例（提要求领域）**：

```
提要求（Mand）是指儿童在有动机的情况下，通过语言、手势、图片、辅助沟通系统或其他沟通方式向他人表达自己的需求。该领域主要反映孩子能否主动、功能性地表达想要的物品、活动、帮助或信息。

根据本次VB-MAPP评估结果，孩子在提要求领域得分为__/15，目前处于Level __阶段。孩子已能够______（基于已得分条目描述，如：在物品可见时自发提出多种不同的要求），说明其已具备一定的基础提要求能力。

但在______条目中尚未达到得分标准（基于未得分条目描述，如：用特殊疑问句提出关于信息的要求、在不同环境中有礼貌地要求停止不喜欢的活动），说明其主动沟通的多样性、请求的复杂度和跨环境泛化能力仍需进一步发展。

后续干预建议围绕当前未得分或部分得分的里程碑条目展开，重点包括提高孩子在自然动机下的主动提要求能力，扩展请求的形式和内容，增加跨人物、跨环境的泛化，并逐步减少辅助。
```

## 5. 障碍评估 — 报告撰写

### 5.1 报告定位

障碍评估报告应说明哪些障碍正在影响孩子的学习效果，以及如何在教学中应对这些障碍。重点不是罗列分数，而是解释障碍对技能习得的具体影响和干预方向。

### 5.2 计分总表

以表格/网格图呈现24项障碍得分。（此部分为数据可视化，不需要AI生成文字）

### 5.3 表后总结

整体障碍情况、高分障碍项、对教学的影响、小结融合为一段连续的专业文本：

```
儿童障碍评估总分为__/96，整体障碍水平为______。

得分最高（最严重）的障碍为______（__分）和______（__分），
这些障碍已显著影响教学效果，需要优先制定干预策略。

得分中等（2-3分）的障碍包括______，需要在日常教学中持续关注。

障碍评估结果提示，当前影响儿童技能发展的核心障碍因素是______，
这与里程碑评估中______领域薄弱的表现相互印证。
后续教学计划应优先处理______等障碍，为技能教学扫清障碍。
```

### 5.4 24项障碍逐项分析

每个障碍的分析必须以**连续、专业、可读的段落**方式呈现，禁止使用列表形式。应说明：该障碍是什么、当前严重程度如何、对教学的具体影响是什么、应如何干预。

**内在结构（不在报告中显式标注，但每段必须覆盖）**：

1. **障碍定义**：该障碍是什么，为什么会影响学习
2. **当前严重程度**：结合得分和评分说明，描述当前表现
3. **对教学的影响**：该障碍如何具体干扰技能习得，与里程碑评估中哪些领域的薄弱相关联
4. **干预建议**：针对该障碍的具体干预策略

**示例（不恰当行为）**：

```
不恰当行为是指儿童在教学或日常情境中出现的干扰学习和社交的行为，包括哭闹、攻击、自伤、破坏物品等。这些行为不仅直接中断教学进程，还会影响儿童与同伴和成人的互动质量。

根据本次评估，该项得分为__分，表明______（基于评分说明描述当前表现，如：孩子每天都发生种种小的负面行为，包括哭闹、语言拒绝、躺到地上等）。这些行为在教学中表现为______，与里程碑评估中______领域的薄弱存在关联——行为问题消耗了大量教学时间，减少了有效学习机会。

后续干预建议包括：进行行为功能评估（FBA）以明确行为的维持变量，制定基于功能的行为干预计划（BIP），同时通过替代行为训练（如功能性沟通训练）减少不恰当行为的发生频率。
```

## 6. 转衔评估 — 计算规则

### 6.1 评分体系

18个领域，每项 0~5 分，得分越高转衔准备越充分。

### 6.2 三大类别

| 类别 | 包含项目 |
|------|---------|
| 第一类：VB-MAPP得分和独立学习能力 | 里程碑总分、障碍总分、负面行为和教学控制、教室常规、社会技能、独立学业工作 |
| 第二类：学习模式 | 泛化、强化物范围、获得技能速度、新技能维持、自然环境学习、语言行为转换 |
| 第三类：自理自发自主 | 对改变的适应性、自发行为、自我导向休闲、一般自理、如厕、进餐 |

### 6.3 自动计算题目（前5项）

| 项目 | 数据来源 | 换算规则 |
|------|---------|---------|
| 里程碑总分 | #1~#170得分之和 | 0-25→1, 26-50→2, 51-100→3, 101-135→4, 136-170→5 |
| 障碍总分 | #171~#194得分之和 | 73-96→1, 49-72→2, 25-48→3, 13-24→4, 0-12→5 |
| 负面行为+教学控制 | #171+#172 | 8→0, 6-7→1, 5→2, 3-4→3, 2→4, 0-1→5 |
| 教室常规 | #136~#145得分之和 | 0-1→0, 2→1, 3-4→2, 5-7→3, 8-9→4, 10→5 |
| 社会技能 | #76~#90得分之和 | 0-1→0, 2-3→1, 4-5→2, 6-9→3, 10-12→4, 13-15→5 |

### 6.4 单项评级

| 得分 | 评级 |
|------|------|
| 0~1 | 需重点关注 |
| 2~3 | 需进一步提升 |
| 4~5 | 已具备转衔准备 |

### 6.5 转衔总体建议

| 总分(满分90) | 建议 |
|-------------|------|
| 0~30 | 以密集个训为主，暂不建议融合环境 |
| 31~60 | 个训为主、小组为辅，逐步增加集体活动 |
| 61~90 | 可考虑融合教育环境，配合适当支持 |

## 7. 转衔评估 — 报告撰写

### 7.1 报告定位

转衔评估报告应说明孩子是否具备从高密度一对一教学逐步过渡到小组教学、自然环境教学或融合环境的准备条件。关注的不是单项技能分数，而是孩子整体的学习独立性、适应性和泛化能力。

### 7.2 计分总表

以表格呈现18项转衔得分，按三大类别分组。（此部分为数据可视化，不需要AI生成文字）

### 7.3 表后总结

整体准备度、优势领域、需支持领域、安置建议、小结融合为一段连续的专业文本：

```
转衔评估主要用于判断孩子是否具备从高密度一对一教学逐步过渡到小组教学、自然环境教学或融合环境的准备条件。该部分关注孩子的学习独立性、问题行为水平、听从集体指令能力、等待能力、转换能力、泛化能力、社会互动能力和对自然强化的反应。

儿童转衔评估总分为__/90，整体转衔准备度为______。

根据本次评估，孩子在______方面表现出一定优势（得分4-5分），例如______，表明儿童已具备______等能力，可作为转衔的支撑点。

但在______方面仍存在困难（得分0-1分），例如______。这些困难提示孩子目前仍需要较高水平的成人支持，在转入更开放或更复杂的教学环境前，应优先提升______能力。

综合里程碑评估（总分__/170）和障碍评估（总分__/96）的结果，当前建议______。后续建议包括逐步增加小组活动参与机会，训练等待、轮流、转换、跟随集体指令等技能，同时减少对一对一成人辅助的依赖。转衔不是一次性决策，而是持续评估和调整的过程。
```

### 7.4 18项转衔领域逐项分析

每个领域的分析必须以**连续、专业、可读的段落**方式呈现，禁止使用列表形式。应说明：该项评估的是什么准备技能、孩子当前的表现如何、对转衔意味着什么、如果准备不足可能出现什么问题、应如何支持。

**内在结构（不在报告中显式标注，但每段必须覆盖）**：

1. **项目说明与得分**：该项评估什么转衔准备技能，当前得分和评级
2. **当前表现**：基于得分和评分说明描述孩子的具体表现
3. **转衔意义**：该能力对进入下一环境意味着什么
4. **风险提示**：准备不足进入新环境后可能出现的问题
5. **支持建议**：具体的提升策略

**示例（泛化能力）**：

```
技能泛化是指儿童能否将在教学环境中习得的技能自然地应用到不同的人物、材料、环境和情境中。这是转衔准备的核心能力之一——如果孩子只能在特定条件下表现出已学技能，进入新环境后将面临严重的适应困难。

根据本次评估，该项得分为__分，表明______（基于评分说明描述）。孩子目前能够______，但在______方面仍存在困难。如果在泛化能力不足的情况下进入集体教学或融合环境，孩子可能出现在新环境中"不会"已学技能、需要重新教学、无法跟上集体节奏等问题。

后续建议在日常教学中系统安排泛化训练，包括变换教学材料、更换教学人员、在不同环境中练习已掌握技能，逐步提高孩子在自然条件下独立运用技能的能力。
```

## 8. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- ABA 专业术语保持准确（提要求=Mand，命名=Tact，听者反应=LR，LRFFC等）
- 写法模板中的"______"必须用实际数据填充，不能留空

### 写作规范

- **段落式呈现**：所有领域分析、障碍分析、转衔分析均以连续专业段落呈现，禁止使用"一、二、三"或"1、2、3"列表
- **基于条目分析**：不能只写"该领域得分较低"或"需要加强"，必须结合具体条目说明会什么、不会什么、差距在哪里
- **结果解读优先**：报告重点是结果解读和干预方向说明，而非分数呈现和评分方法解释
- **干预建议必须对应未掌握条目**：每个领域的干预建议必须与该领域未得分或部分得分的具体条目相对应
- **面向家长可读**：家长不需要理解分数如何计算，需要理解孩子会什么、不会什么、为什么要设这些目标、怎么干预
