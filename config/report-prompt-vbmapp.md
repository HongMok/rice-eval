- **1分（达标）**：完全达到该里程碑
- **0.5分（部分达标）**：部分达到
- **0分（未达标）**：未达到

### 2.5 得分计算

```
里程碑评估得分 = 所有里程碑题目的得分之和

Level 1 得分 = Σ(Level 1 所有领域所有里程碑的得分)
Level 2 得分 = Σ(Level 2 所有领域所有里程碑的得分)
Level 3 得分 = Σ(Level 3 所有领域所有里程碑的得分)

里程碑评估得分 = Level 1 得分 + Level 2 得分 + Level 3 得分
满分 = 170分
```

**各 Level 满分明细**

```
Level 1：9个领域 × 5个里程碑 × 1分 = 45分
Level 2：12个领域 × 5个里程碑 × 1分 = 60分（新增 LRFFC、对话、集体技能、语言结构，去掉发音）
Level 3：13个领域 × 5个里程碑 × 1分 = 65分（新增数学、阅读、书写，去掉模仿、仿说）
总计：45 + 60 + 65 = 170分
```

**领域得分**

```
领域得分 = 该领域在所有出现的 Level 中的得分之和
领域得分率 = 领域得分 / 领域满分 × 100%

例：提要求（跨3个Level）
  Level 1: 1M~5M 得分之和（满分5）
  Level 2: 6M~10M 得分之和（满分5）
  Level 3: 11M~15M 得分之和（满分5）
  提要求总分 = Level 1 + Level 2 + Level 3（满分15）

例：发音（仅 Level 1）→ 满分5
例：数学（仅 Level 3）→ 满分5
```

| 得分率 | 评级 |
|--------|------|
| ≥ 70% | 表现良好 |

### 2.7 发展阶段判定

| 总分范围 | 发展阶段 |
|---------|---------|
| 0~45 | 第一阶段水平（0-18月） |
| 46~90 | 第二阶段水平（18-30月） |
| 91~170 | 第三阶段水平（30-48月） |

### 2.8 里程碑评估报告（每个领域）

每个领域的报告卡片包含：
- **领域名称**（中英文）+ 满分 + 得分
- **IEP目标（未来三个月）**：根据该领域当前得分阶段，列出下一阶段的具体教学目标（5条左右）
- 目标内容必须基于 VB-MAPP 原始量表的里程碑描述，确定性生成

### 2.9 里程碑评估建议

在所有领域报告之后，生成一段综合分析文本，撰写思路：
1. **总体概述** — 孩子的评估情况一览，优势领域、评估中的亮点
2. **分项描述** — 不同领域出现的问题，结合数据举出实例
3. **给出建议** — 结合里程碑得分，给出改善建议，包含训练方向+举例+步骤
4. **结语** — 列出需要训练项目的优先级，并给出理由

| 2~3 | 需进一步提升，部分具备转衔准备 |
| 4~5 | 表现良好，已具备转衔准备 |

### 4.6 转衔总体建议

```
转衔总分 = 18项得分之和（满分90分）
```

| 总分范围 | 建议 |
|---------|------|
| 0~30 | 以密集个训为主，暂不建议融合环境 |
| 31~60 | 个训为主、小组为辅，逐步增加集体活动 |
| 61~90 | 可考虑融合教育环境，配合适当支持 |

### 4.7 转衔评估报告（每个项目）

每个转衔项的报告卡片包含：
- **项目名称**（中英文）+ 得分
- **描述**：就该单项而言的得分解读
- **阶段性转衔安置建议**：基于得分给出的安置建议

### 4.8 转衔评估建议

撰写思路同里程碑评估建议（总体概述→分项描述→给出建议→结语）。

## 5. 评估建议撰写规则

三个模块（里程碑、障碍、转衔）各有一段"评估建议"文本，统一撰写思路：

```
1. 总体概述 — 孩子的评估情况一览（优势领域、亮点、当前水平）
2. 分项描述 — 描述不同领域出现的问题或需要让家长了解的情况
   - 结合数据，举出实例
   - 每一个"问题"的描述后，都要举出评估中真实出现的实例
3. 给出建议 — 根据评估中出现的问题，结合得分，给出改善建议
   - "建议"中不仅要包含训练方向
   - 还应适当做出举例便于家长理解
   - 或是给出明确步骤方便家长实际操作
4. 结语 — 最后小结，列出需要训练项目的优先级，并给出理由
```

## 6. 综合分析生成规则

| 模块 | 内容规则 |
|------|---------|
| 概述 | 必须包含三大模块的总分和发展阶段判定 |
| 优势领域 | 里程碑得分率 ≥ 70% 的领域 + 转衔得分 ≥ 4 的项目 |
| 需关注领域 | 里程碑得分率 < 40% 的领域 + 障碍得分 ≥ 3 的项目 |
| 训练建议 | 按优先级：P1=障碍得分≥3 → P2=里程碑得分率<40% → P3=转衔得分≤1 |

## 7. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- ABA 专业术语保持准确（提要求=Mand，命名=Tact，听者反应=Listener Responding，LRFFC=Listener Responding by Function Feature and Class 等）
