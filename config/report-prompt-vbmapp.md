# VB-MAPP 评估报告 — 计算规则

> 本文档定义 VB-MAPP（语言行为里程碑评估和安置程序）评估报告的所有计算规则。
> 同一份评估数据，必须产出完全相同的报告结果。

---

## 1. 评估结构

VB-MAPP 包含三大评估模块，采用三色主题区分：

| 模块 | 主题色 | 满分 | 说明 |
|------|--------|------|------|
| 里程碑评估 | 🟢 绿色 | 170分 | 发展性技能评估，3个阶段 × 多领域 |
| 障碍评估 | 🟠 橙色 | 96分 | 24项学习障碍评估，得分越高障碍越严重 |
| 转衔评估 | 🔵 蓝色 | 90分 | 18个领域，评估转衔准备程度 |

---

## 2. 里程碑评估

### 2.1 评分体系

每个里程碑三级评分：
- **1分（达标）**：完全达到该里程碑
- **0.5分（部分达标）**：部分达到
- **0分（未达标）**：未达到

### 2.2 三级阶段结构

| 阶段 | 月龄范围 | 说明 |
|------|---------|------|
| Level 1 | 0-18月 | 基础阶段 |
| Level 2 | 18-30月 | 中级阶段 |
| Level 3 | 30-48月 | 高级阶段 |

每领域每阶段 5 个里程碑，满分 5 分。

### 2.3 里程碑领域（16个）

```
提要求、命名、听者反应、视觉配对、游戏、社交、模仿、仿说、
发音、LRFFC、对话、集体技能、语言结构、数学、阅读、书写
```

### 2.4 得分计算

```
领域得分 = Level 1 得分 + Level 2 得分 + Level 3 得分
领域得分率 = 领域得分 / 领域满分 × 100%
里程碑总分 = 所有领域得分之和
```

### 2.5 领域评级

| 得分率 | 评级 |
|--------|------|
| ≥ 70% | 表现良好 |
| 40%~69% | 发展中 |
| < 40% | 需重点关注 |

### 2.6 发展阶段判定

基于里程碑总分：

| 总分范围 | 发展阶段 |
|---------|---------|
| 0~45 | 第一阶段水平（0-18月） |
| 46~90 | 第二阶段水平（18-30月） |
| 91~170 | 第三阶段水平（30-48月） |

### 2.7 里程碑网格填色

| 得分 | 填色 |
|------|------|
| 1 | 全填色（cell-filled） |
| 0.5 | 半填色（cell-half） |
| 0 | 空白（cell-empty） |

## 3. 障碍评估

### 3.1 评分体系

24项障碍，每项 0~4 分，得分越高障碍越严重。

### 3.2 障碍项目（24项）

```
行为问题、教学控制、提要求的缺陷、命名的缺陷、模仿的缺陷、
仿说的缺陷、视觉感知和配对的缺失、听者技能的缺陷、对话的缺陷、
社会技能的缺陷、依赖辅助、猜想式回答、扫视的缺陷、
条件性辨别的缺陷、不能泛化、薄弱动机、对行为有要求就会减弱动机、
依赖强化物、自我刺激、发音清晰度的缺陷、强迫性行为、
多动行为、没有目光接触、感觉防御
```

### 3.3 单项严重度

| 得分 | 严重度 |
|------|--------|
| 0 | 无明显障碍 |
| 1 | 轻微障碍，影响较小 |
| 2 | 中等障碍，需关注 |
| 3 | 较严重障碍，需优先干预 |
| 4 | 严重障碍，已显著影响学习 |

### 3.4 障碍总体评级

```
障碍总分 = 24项得分之和（满分96分）
```

| 总分范围 | 评级 |
|---------|------|
| 0~24 | 整体障碍水平较低 |
| 25~48 | 存在一定障碍，需在教学中注意 |
| 49~72 | 障碍较为显著，需制定针对性干预 |
| 73~96 | 障碍严重，需优先全面干预 |

### 3.5 重点关注障碍

```
topConcerns = 得分 ≥ 3 的障碍项，按得分降序排列
```

## 4. 转衔评估

### 4.1 评分体系

18个领域，每项 0~5 分，得分越高转衔准备越充分。

### 4.2 三大类别

| 类别 | 包含领域 |
|------|---------|
| 第一类：VB-MAPP得分和独立学习的能力 | VB-MAPP里程碑得分、VB-MAPP障碍得分、负面行为和教学控制、教室常规、集体技能、社会技能和社会游戏、独立的学业工作 |
| 第二类：学习模式 | 泛化、强化物泛化、获得技能的速度、新技能的维持、自然环境中的学习、未经训练的转换 |
| 第三类：语言和社交技能 | 自发语言、独立的对话、社交互动、同伴互动、社区活动 |

### 4.3 单项评级

| 得分 | 评级 |
|------|------|
| 0~1 | 需重点关注，尚不具备转衔准备 |
| 2~3 | 需进一步提升，部分具备转衔准备 |
| 4~5 | 表现良好，已具备转衔准备 |

### 4.4 转衔总体建议

```
转衔总分 = 18项得分之和（满分90分）
```

| 总分范围 | 建议 |
|---------|------|
| 0~30 | 以密集个训为主，暂不建议融合环境 |
| 31~60 | 个训为主、小组为辅，逐步增加集体活动 |
| 61~90 | 可考虑融合教育环境，配合适当支持 |

## 5. 图表

### 5.1 里程碑计分总表（网格图）

```
结构：3个阶段（Level 3 在上，Level 1 在下），每阶段按领域列排列
每格 = 一个里程碑
填色规则：
  得分 1   → 全填色（主题绿色 #43A047）
  得分 0.5 → 半填色（浅绿 #A5D6A7）
  得分 0   → 空白（#F0F2F5）
```

### 5.2 障碍计分总表（网格图）

```
结构：24项障碍，每项4格（对应0~4分）
填色规则：
  从左到右填色格数 = 该项得分
  填色 → 主题橙色 #F9A825
  空白 → #F0F2F5
```

### 5.3 转衔计分总表（网格图）

```
结构：按三大类别分组，每项5格（对应0~5分）
填色规则：
  从左到右填色格数 = 该项得分
  填色 → 主题蓝色 #1E88E5
  空白 → #F0F2F5
```

### 5.4 领域报告卡片

```
每个领域一张卡片，包含：
  领域名 + 得分/满分
  评价描述（基于 2.5 领域评级规则生成）
  IEP建议（基于评级生成确定性建议文本）

卡片左侧色条：
  里程碑领域 → 绿色 #43A047
  障碍项 → 橙色 #F9A825
  转衔领域 → 蓝色 #1E88E5
```

## 6. 综合分析生成规则

| 模块 | 内容规则 |
|------|---------|
| 概述 | 必须包含三大模块的总分和发展阶段判定 |
| 优势领域 | 里程碑得分率 ≥ 70% 的领域 + 转衔得分 ≥ 4 的项目 |
| 需关注领域 | 里程碑得分率 < 40% 的领域 + 障碍得分 ≥ 3 的项目 |
| 训练建议 | 按优先级：P1=障碍得分≥3 → P2=里程碑得分率<40% → P3=转衔得分≤1 |

---

## 7. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- ABA 专业术语保持准确（提要求=Mand，命名=Tact 等）
