# VB-MAPP 评估报告 — 计算规则与撰写模板

> 本文档定义 VB-MAPP 评估报告的所有计算规则和撰写模板。
> 同一份评估数据，必须产出完全相同的报告结果。

---

## 报告整体结构

```
基本信息（姓名/性别/出生日期/测评日期）

第一部分：里程碑评估
  ├─ 计分总表（网格图）
  ├─ 表后总结（整体情况+优势+薄弱+小结，一段完整文本）
  └─ 16个领域逐项分析

第二部分：障碍评估
  ├─ 计分总表
  ├─ 表后总结（整体情况+高分障碍+对教学的影响+小结）
  └─ 24项障碍逐项分析

第三部分：转衔评估
  ├─ 计分总表
  ├─ 表后总结（整体准备度+优势+需支持领域+安置建议+小结）
  └─ 18项转衔领域逐项分析
```

## 2. 里程碑评估 — 计算规则

### 2.1 评分体系

1分（达标）、0.5分（部分达标）、0分（未达标）

### 2.2 三级阶段

| 阶段 | 月龄 | 满分 |
|------|------|------|
| Level 1 | 0-18月 | 45 |
| Level 2 | 18-30月 | 60 |
| Level 3 | 30-48月 | 65 |

### 2.3 16个领域满分表

| 领域 | L1 | L2 | L3 | 满分 |
|------|----|----|----|----|
| 提要求(Mand) | 5 | 5 | 5 | 15 |
| 命名(Tact) | 5 | 5 | 5 | 15 |
| 听者反应(LR) | 5 | 5 | 5 | 15 |
| 视觉配对(VP-MTS) | 5 | 5 | 5 | 15 |
| 游戏(Play) | 5 | 5 | - | 10 |
| 社交(Social) | 5 | 5 | 5 | 15 |
| 模仿(Imitation) | 5 | 5 | - | 10 |
| 仿说(Echoic) | 5 | 5 | - | 10 |
| 发音(SVB) | 5 | - | - | 5 |
| LRFFC | - | 5 | 5 | 10 |
| 对话(IV) | - | 5 | 5 | 10 |
| 集体技能(Group) | - | 5 | 5 | 10 |
| 语言结构(Ling) | - | 5 | 5 | 10 |
| 数学(Math) | - | - | 5 | 5 |
| 阅读(Reading) | - | - | 5 | 5 |
| 书写(Writing) | - | - | 5 | 5 |
| **合计** | **45** | **60** | **65** | **170** |

### 2.4 得分计算

```
领域得分 = 该领域在所有出现的Level中的得分之和
领域得分率 = 领域得分 / 领域满分 × 100%
里程碑总分 = 所有领域得分之和（满分170）
```

### 2.5 发展阶段判定

| 总分 | 阶段 |
|------|------|
| 0~45 | Level 1（0-18月） |
| 46~90 | Level 2（18-30月） |
| 91~170 | Level 3（30-48月） |

### 2.6 领域评级

| 得分率 | 评级 |
|--------|------|
| ≥ 70% | 表现良好 |
| 40%~69% | 发展中 |
| < 40% | 需重点关注 |

## 3. 里程碑评估 — 报告撰写

### 3.1 计分总表

以网格图形式呈现所有领域在三个等级的得分。

### 3.2 表后总结

整体情况、优势领域、薄弱领域、评估小结融合为一段完整文本：

```
儿童当前实际年龄为X岁X个月，VB-MAPP 里程碑总分为 __/170，
技能主要集中在 Level __（大致对应__至__个月发展水平）。
其已具备______等基础能力，但在______等领域仍需系统教学。

从各领域得分看，相对优势领域为______（得分率___%）和______（得分率___%），
这些优势是可以利用的行为库，教学设计应利用优势作为切入点支持薄弱领域。

当前明显薄弱的领域为______（得分率___%）和______（得分率___%），
需要区分技能缺陷和表现缺陷，优先安排系统教学。

综合来看，各领域发展并不均衡，尤其是______与______之间的差距提示______。
后续教学应结合领域间关系进行课程设计，
同时需要结合障碍评估结果分析影响技能发展的障碍因素。
```

### 3.3 各领域逐项分析（16个领域）

每个领域使用统一的**七项结构**：

```
一、领域定义与评估目的
    用 Skinner 语言行为理论术语定义该领域

二、得分情况
    得分/满分，处于哪个里程碑阶段

三、当前发展水平描述
    基于数据描述儿童当前能做什么

四、优势表现
    该领域中已具备的能力（具体到可观察行为）

五、主要困难
    该领域中的主要问题

六、教学意义
    评估结果对教学计划意味着什么

七、建议方向
    具体的教学策略建议
```

## 6. 转衔评估 — 计算规则

### 6.1 评分体系

18个领域，每项 0~5 分，得分越高转衔准备越充分。

### 6.2 三大类别

| 类别 | 包含项目 |
|------|---------|
| 第一类：VB-MAPP得分和独立学习能力 | 里程碑总分、障碍总分、负面行为和教学控制、教室常规、社会技能、独立学业工作 |
| 第二类：学习模式 | 泛化、强化物范围、获得技能速度、新技能维持、自然环境学习、语言行为转换 |
| 第三类：自理自发自主 | 对改变的适应性、自发行为、自我导向休闲、一般自理、如厕、进餐 |

### 6.3 自动计算题目（前5项）

| 项目 | 数据来源 | 换算规则 |
|------|---------|---------|
| 里程碑总分 | #1~#170得分之和 | 0-25→1, 26-50→2, 51-100→3, 101-135→4, 136-170→5 |
| 障碍总分 | #171~#194得分之和 | 73-96→1, 49-72→2, 25-48→3, 13-24→4, 0-12→5 |
| 负面行为+教学控制 | #171+#172 | 8→0, 6-7→1, 5→2, 3-4→3, 2→4, 0-1→5 |
| 教室常规 | #136~#145得分之和 | 0-1→0, 2→1, 3-4→2, 5-7→3, 8-9→4, 10→5 |
| 社会技能 | #76~#90得分之和 | 0-1→0, 2-3→1, 4-5→2, 6-9→3, 10-12→4, 13-15→5 |

### 6.4 单项评级

| 得分 | 评级 |
|------|------|
| 0~1 | 需重点关注 |
| 2~3 | 需进一步提升 |
| 4~5 | 已具备转衔准备 |

### 6.5 转衔总体建议

| 总分(满分90) | 建议 |
|-------------|------|
| 0~30 | 以密集个训为主，暂不建议融合环境 |
| 31~60 | 个训为主、小组为辅，逐步增加集体活动 |
| 61~90 | 可考虑融合教育环境，配合适当支持 |

## 8. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- ABA 专业术语保持准确（提要求=Mand，命名=Tact，听者反应=LR，LRFFC等）
- 写法模板中的"______"必须用实际数据填充，不能留空
- 里程碑逐项分析：七项结构
- 障碍逐项分析：五项结构
- 转衔逐项分析：五项结构
