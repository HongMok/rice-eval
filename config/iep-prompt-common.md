# 通用评估结果 → IEP 生成规则

> 本文档定义用户自建评估工具的 IEP 生成规则。
> 核心原则：IEP 中的所有内容必须来自课包干预项目配置表，不能自行编造。

---

## 1. IEP 表格结构

| 列 | 数据来源 |
|----|---------|
| 阶梯及领域 | `{年龄段}-{一级领域}-{二级领域}` |
| 任务目标 | 干预项目名称（`intervene_project.name`） |
| 短期目标 | 干预项目阶段名称（`intervene_project_stage.name`） |

---

## 2. 生成规则

1. **任务目标 = 干预项目名称**，颗粒度细到具体项目
2. **短期目标 = 干预项目阶段名称**，按阶段顺序排列
3. **不匹配时跳过**，**不能编造**
4. **导出文件命名**：`{姓名}个别化教育计划.docx`
