## 3. 副测验分数计算

### 3.1 原积分（Raw Score）

```
原积分 = 该副测验所有题目中得分为 2（通过/恰当）的题目数量

注意：只计 2 分的题目数，不是总得分。
萌芽(1分)和不通过(0分)都不计入原积分。
```

**计算示例**：
```
CVP 副测验共 35 题
  得 2 分的题目：20 题
  得 1 分的题目：8 题
  得 0 分的题目：7 题
  原积分 = 20
```

### 3.2 发展年龄（Developmental Age）

通过原积分查**发展表现图常模表**得到发展年龄（月数）。

常模表是一张二维查找表：
- Y 轴 = 年龄（月），范围 <12 ~ 83
- X 轴 = 副测验代码（CVP, EL, RL, FM, GM, VMI, PSC）
- 表格中的数值 = 该副测验在该年龄对应的原积分

**查表方法**：
```
1. 取该副测验的原积分
2. 在常模表的该副测验列中，找到原积分对应的年龄行
3. 如果原积分恰好在表中 → 直接取该行年龄
4. 如果原积分在两行之间 → 取较低年龄行（向下取整）
5. 发展年龄 = 查到的年龄（月）
```

**注意**：
- 只有发展副测验（CVP, EL, RL, FM, GM, VMI）和照顾者报告中的 PSC 有发展年龄
- 行为副测验（AE, SR, CMB, CVB）和照顾者报告中的 PB、AB 没有发展年龄，只有适应程度

### 3.3 百分比级数（Percentile Rank）

通过原积分查**百分比级数常模表**得到。

**简化映射规则**（当无法获取完整常模表时使用）：

```
得分率 = 原积分 / 该副测验总题数 × 100%
```

| 得分率 | 百分比级数 |
|--------|-----------|
| ≥ 90% | 95 |
| 80%~89% | 75 |
| 70%~79% | 50 |
| 60%~69% | 25 |
| 50%~59% | 10 |
| < 50% | 5 |

### 3.4 发展/适应程度（Developmental/Adaptive Level）

| 百分比级数 | 程度 |
|-----------|------|
| ≥ 75 | 适当（Appropriate） |
| 25 ~ 74 | 轻度（Mild） |
| < 25 | 严重（Severe） |

## 4. 合成分数计算

### 4.1 三个合成分数的组成

| 合成分数 | 代码 | 组成副测验 |
|---------|------|-----------|
| 沟通 | C | CVP + EL + RL |
| 体能 | M | FM + GM + VMI |
| 行为 | MB | AE + SR + CMB + CVB |

### 4.2 计算步骤

```
Step 1: 取各组成副测验的标准分（百分比级数）
Step 2: 标准分总和 = 各组成副测验的百分比级数之和
Step 3: 查合成分数常模表 → 合成百分比级数
Step 4: 合成发展/适应程度 = 同 3.4 的映射规则
Step 5: 合成发展年龄 = 各组成副测验发展年龄的平均值（仅沟通C和体能M有）
```

**计算示例**：
```
沟通(C) = CVP + EL + RL
  CVP 百分比级数 = 50
  EL 百分比级数 = 25
  RL 百分比级数 = 25
  标准分总和 = 50 + 25 + 25 = 100
  → 查合成常模表得合成百分比级数
  → 映射发展/适应程度

体能(M) = FM + GM + VMI
  FM 百分比级数 = 10
  GM 百分比级数 = 50
  VMI 百分比级数 = 75
  标准分总和 = 10 + 50 + 75 = 135
  → 查合成常模表得合成百分比级数

行为(MB) = AE + SR + CMB + CVB
  AE 百分比级数 = 50
  SR 百分比级数 = 25
  CMB 百分比级数 = 75
  CVB 百分比级数 = 50
  标准分总和 = 50 + 25 + 75 + 50 = 200
  → 查合成常模表得合成百分比级数
```

## 5. 功能发展评分表（P/E/F 体系）

### 5.1 统计方法

对每个发展副测验，统计三种评分的题目数：

```
P（通过）题数 = 得分为 2 的题目数
E（萌芽）题数 = 得分为 1 的题目数
F（不通过）题数 = 得分为 0 的题目数
总题数 = P + E + F
```

### 5.2 得分率

```
得分率 = P题数 / 总题数 × 100%
```

### 5.3 领域评价

| 条件 | 评价 |
|------|------|
| 得分率 ≥ 75% | 强处 🟢 |
| 得分率 ≤ 40% | 弱处 🔴 |
| 其余 | 不标注 |

### 5.4 整体发展年龄

```
整体发展年龄 = 所有有发展年龄的副测验（CVP, EL, RL, FM, GM, VMI, PSC）的发展年龄中位数
```

## 6. 行为表现评分

### 6.1 评级规则

对每个行为副测验（AE, SR, CMB, CVB），计算领域内平均分：

```
领域平均分 = 领域总得分 / 领域总题数
```

| 平均分 | 评级 | 含义 |
|--------|------|------|
| ≥ 1.5 | A（适当） | 行为表现在正常范围 |
| 0.8 ~ 1.49 | M（轻度） | 存在轻度异常 |
| < 0.8 | S（严重） | 存在严重异常 |

### 6.2 计算示例

```
情感表达(AE) 共 11 题
  总得分 = 16（各题得分之和）
  平均分 = 16 / 11 = 1.45
  评级 = M（轻度）
```

## 8. 综合分析生成规则

综合分析文本的结构和内容规则：

### 7.1 各领域得分概述

```
格式：本次评估涵盖功能发展{N1}个领域和行为表现{N2}个领域。
功能发展总分{总题数}分中，已有能力(P){P总数}分，部分能力(E){E总数}分，未达到(F){F总数}分。
整体发展年龄{X}岁{Y}月。
```

### 7.2 优势领域

```
列出所有得分率 ≥ 60% 的发展副测验，按得分率降序
格式：{领域名}({得分率}%)
```

### 7.3 需关注领域

```
列出所有得分率 ≤ 40% 的发展副测验，按得分率升序
格式：{领域名}({得分率}%)，得分率偏低，建议作为重点干预领域。
```

### 7.4 行为表现

```
列出所有行为副测验的评级
格式：{领域名}({评级}-{含义})
```

### 7.5 建议

```
结合弱处领域和行为异常，给出训练建议
优先级：P1=S级行为 → P2=弱处领域 → P3=M级行为
```

## 9. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- 原积分只计 2 分（通过/恰当）的题目数，不是总得分
