# PEP-3 评估报告 — 计算规则

> 本文档定义 PEP-3 评估报告的所有计算规则。
> 同一份评估数据，必须产出完全相同的报告结果。所有判断基于下方量化阈值，不允许随机或模糊推导。

---

## 1. 评估结构

PEP-3 包含两大板块：

| 板块 | 领域数 | 领域列表 |
|------|--------|---------|
| 功能发展 | 7 | 模仿(I)、知觉(P)、小肌肉(FM)、大肌肉(GM)、手眼协调(EH)、认知理解(CP)、认知表达(CV) |
| 行为表现 | 4 | 情感关系(R)、游戏及兴趣(M)、感觉反应(S)、语言异常(L) |

---

## 2. 功能发展评分计算

### 2.1 原始评分体系

每题三级评分：
- **P（通过）= 2分**：独立完成
- **E（萌芽）= 1分**：部分完成或需要提示
- **F（不通过）= 0分**：无反应或无法完成

### 2.2 得分率

```
得分率 = P题数 / 总题数 × 100%
```

四舍五入取整。

### 2.3 领域评价（强处/弱处）

| 条件 | 评价 |
|------|------|
| 得分率 ≥ 75% | 强处 🟢 |
| 得分率 ≤ 50% | 弱处 🔴 |
| 其余 | 不标注 |

### 2.4 发展年龄推算

基于得分率和儿童实足年龄（月数），确定性映射：

| 得分率 | 发展年龄 |
|--------|---------|
| ≥ 90% | 实足年龄 |
| 80%~89% | 实足年龄 - 6个月 |
| 70%~79% | 实足年龄 - 12个月 |
| 60%~69% | 实足年龄 - 18个月 |
| 50%~59% | 实足年龄 - 24个月 |
| 40%~49% | 实足年龄 - 30个月 |
| < 40% | 实足年龄 - 36个月 |

发展年龄最低不低于 0个月。

### 2.5 整体发展年龄

```
整体发展年龄 = 7个领域发展年龄的中位数
```

---

## 3. 行为表现评分计算

### 3.1 评级规则

每题评分 0~2 分，计算领域内平均分：

```
领域平均分 = 领域总得分 / 领域总题数
```

| 平均分 | 评级 | 含义 |
|--------|------|------|
| ≥ 1.5 | A（适当） | 行为表现在正常范围 |
| 0.8 ~ 1.49 | M（轻度） | 存在轻度异常 |
| < 0.8 | S（严重） | 存在严重异常 |

---

## 4. 副测验分数表

### 4.1 原始分

```
原始分 = P题数（通过的题目数）
```

### 4.2 百分比级数

基于得分率的确定性映射：

| 得分率 | 百分比级数 |
|--------|-----------|
| ≥ 90% | 95 |
| 80%~89% | 75 |
| 70%~79% | 50 |
| 60%~69% | 25 |
| 50%~59% | 10 |
| < 50% | 5 |

### 4.3 发展/适应程度

| 百分比级数 | 程度 |
|-----------|------|
| ≥ 75 | 适当 |
| 50~74 | 轻度 |
| < 50 | 严重 |

---

## 5. 合成分数表

### 5.1 合成分数组成

| 合成分数 | 组成子测验 |
|---------|-----------|
| 沟通(C) | 认知表达(CV) + 认知理解(CP) + 模仿(I) |
| 运动(Mo) | 小肌肉(FM) + 大肌肉(GM) + 手眼协调(EH) |
| 适应行为(MB) | 情感关系(R) + 游戏兴趣(M) + 感觉反应(S) + 语言异常(L) |

### 5.2 计算方法

```
合成标准分总和 = 各组成子测验的百分比级数之和
合成百分比级数 = 合成标准分总和 / 合成满分 × 100，映射到百分比级数表
合成发展/适应程度 = 同 4.3 的映射规则
```

## 6. 发展表现图

折线图，展示各副测验的发展/适应程度等级。

**坐标系**

```
X轴：7个功能发展副测验（CV、CP、I、FM、GM、EH、P）
Y轴：发展/适应程度等级（4级）
  4 = 适当（百分比级数 ≥ 75）
  3 = 轻度（百分比级数 50~74）
  2 = 中度（百分比级数 25~49）
  1 = 严重（百分比级数 < 25）
```

**数据点**

```
每个副测验的 Y 值 = 该副测验的发展/适应程度等级（1~4）
折线连接各点
```

## 7. 雷达图

```
7个功能发展领域的得分率作为雷达图各轴数值
数值范围 0%~100%
各轴值 = 该领域 P题数 / 总题数 × 100%
```

## 8. 综合分析生成规则

| 模块 | 内容规则 |
|------|---------|
| 概述 | 必须包含功能发展总分（P/E/F）和行为表现评级汇总 |
| 优势领域 | 列出所有"强处"领域（得分率 ≥ 75%），按得分率降序 |
| 需关注领域 | 列出所有"弱处"领域（得分率 ≤ 50%）+ S级行为领域，按严重程度排序 |
| 训练建议 | 按优先级：P1=S级行为 → P2=弱处领域 → P3=M级行为 → P4=中等领域 |

---

## 9. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
