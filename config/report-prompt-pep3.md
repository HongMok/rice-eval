# PEP-3 评估报告 — 计算规则

> 本文档定义 PEP-3 评估报告的所有计算规则。
> 同一份评估数据，必须产出完全相同的报告结果。

## 3. 副测验分数计算

### 3.1 原积分（Raw Score）

```
原积分 = 该副测验中所有题目得分之和
发展副测验：每题 0(不通过) / 1(萌芽) / 2(通过)，原积分 = Σ各题得分
行为副测验：每题 0(严重) / 1(轻微) / 2(恰当)，原积分 = Σ各题得分
照顾者报告：每题 0/1/2，原积分 = Σ各题得分
```

### 3.2 发展年龄（Developmental Age）

通过原积分查**附件3：发展副测验原积分和发展年龄对照表**得到发展年龄（月数）。

```
查表方法：
1. 取该副测验的原积分
2. 在附件3对照表的该副测验列中，找到原积分对应的年龄行
3. 原积分恰好在表中 → 直接取该行年龄
4. 原积分在两行之间 → 取较低年龄行（向下取整）
5. 发展年龄 = 查到的年龄（月）
```

适用范围：
- 发展副测验（CVP, EL, RL, FM, GM, VMI）：有发展年龄
- 照顾者报告-PSC：有发展年龄
- 行为副测验（AE, SR, CMB, CVB）和 PB、AB：无发展年龄，只有适应程度

**无常模表时的简化公式**：
```
发展年龄（月）= (原积分 / 该副测验满分) × 儿童实足年龄（月）
```

### 3.3 百分比级数（Percentile Rank）和标准分（Standard Score）

通过原积分 + 儿童年龄段，查**附件1：PEP-3 副测验原积分、标准分和百分比级数对照表**（附表1.1~1.11，按年龄段分11张表）。

```
查表方法：
1. 根据儿童实足年龄确定使用哪张附表：
   附表1.1  = 2岁0月~2岁5月（24-29月龄）
   附表1.2  = 2岁6月~3岁0月（30-36月龄）
   附表1.3  = 3岁0月~3岁5月（36-41月龄）
   附表1.4  = 3岁6月~4岁0月（42-48月龄）
   附表1.5  = 4岁0月~4岁5月（48-53月龄）
   附表1.6  = 4岁6月~5岁0月（54-60月龄）
   附表1.7  = 5岁0月~5岁5月（60-65月龄）
   附表1.8  = 5岁6月~6岁0月（66-72月龄）
   附表1.9  = 6岁0月~6岁5月（72-77月龄）
   附表1.10 = 6岁6月~7岁0月（78-84月龄）
   附表1.11 = 7岁0月~7岁5月（84-89月龄）

2. 在对应附表中，用原积分查该副测验的标准分和百分比级数
3. 标准分用于计算合成分数（见第4节）
4. 百分比级数用于确定发展/适应程度（见3.4）
```

常模数据文件位于 `config/pep3/csv/` 目录下，命名格式 `norm-table-{附表号}-age-{起始年龄}-{结束年龄}.csv`。

**无常模表时的简化映射**：

```
得分率 = 原积分 / 该副测验满分 × 100%
```

| 得分率 | 近似百分比级数 |
|--------|-------------|
| ≥ 90% | 95 |
| 80%~89% | 75 |
| 70%~79% | 50 |
| 60%~69% | 25 |
| 50%~59% | 10 |
| < 50% | 5 |

### 3.4 发展/适应程度

根据百分比级数确定发展/适应程度（依据 PEP-3 测试员手册表3.1）：

| 百分比级数 | 发展/适应程度 | 显示 |
|-----------|-------------|------|
| > 89 | 恰当 | 绿色标签 |
| 75 ~ 89 | 轻微 | 黄色标签 |
| 25 ~ 74 | 中度 | 橙色标签 |
| < 25 | 严重 | 红色标签 |

## 4. 合成分数计算

### 4.1 三个合成分数

| 合成分数 | 代码 | 组成副测验 |
|---------|------|-----------|
| 沟通 | C | CVP + EL + RL |
| 体能 | M | FM + GM + VMI |
| 行为 | MB | AE + SR + CMB + CVB |

### 4.2 计算步骤

```
Step 1: 从附件1对应年龄段的附表中，查出每个副测验的标准分
Step 2: 标准分总和 = 该合成分数内各组成副测验的标准分之和
Step 3: 合成百分比级数 = 用标准分总和查附件2（标准分总和及百分比级数对照表）
Step 4: 合成发展/适应程度 = 同 3.4 的映射规则
Step 5: 合成发展年龄 = 各组成副测验发展年龄的平均值（仅沟通C和体能M有）
```

**附件2对照表说明**：
- 沟通(C)：标准分总和范围 3~49+，对应百分比级数
- 体能(M)：标准分总和范围 3~42+，对应百分比级数
- 行为(MB)：标准分总和范围 4~61+，对应百分比级数

### 4.3 合成分数表格

```
                标准分 Standard Scores (SSs)
                CVP  EL  RL  FM  GM  VMI  AE  SR  CMB  CVB  标准分总和  百分比级数  发展/适应程度  发展年龄
沟通(C)          {分} {分} {分}                                    {和}      {X%}      {程度}       {X月}
体能(M)                        {分} {分} {分}                      {和}      {X%}      {程度}       {X月}
行为(MB)                                      {分} {分} {分} {分}  {和}      {X%}      {程度}       —
```

## 5. 发展表现图（常模侧面图）

### 5.1 图表结构

- X轴 = 7列：CVP、EL、RL、FM、GM、VMI、PSC
- Y轴 = 年龄（月），范围 <12 ~ 83月
- 背景 = 常模表数值（粉色条纹行）

### 5.2 两条数据线

**● 通过项目分数（P分，深绿色实线）**

```
每个副测验的 P 值 = 该副测验中得分为 2 的题目数
在常模表中找到 P 值对应的年龄行 → 画点连线
```

**■ 部分通过项目分数（P+E分，浅绿色实线）**

```
每个副测验的 P+E 值 = 得分为 2 的题目数 + 得分为 1 的题目数
在常模表中找到 P+E 值对应的年龄行 → 画点连线
```

### 5.3 P+E 与 P 的差距 = 萌芽技能

```
P+E 线与 P 线之间的差距 = 萌芽技能数量
差距越大 → 该领域的最近发展区越宽 → 干预潜力越大
```

### 5.4 底部汇总

```
通过项目分数：{各副测验P值}
部分通过项目分数：{各副测验P+E值}
```

## 6. 发展数据分析（水平柱状图）

### 6.1 图表结构

- 6个发展副测验，按得分率从低到高排列（最弱在最上面）
- 每个副测验一行水平柱状图
- 深蓝色 = P（通过）占比
- 浅蓝色 = E（萌芽）占比
- 红色虚线 = 65% 薄弱线（在最上层，不被柱子遮挡）

### 6.2 得分率计算

```
得分率 = P题数 / 总题数 × 100%

P题数 = 得分为 2（通过）的题目数量。
只计算完全通过的题目占比，用于判断领域薄弱程度。
萌芽(E)不计入得分率，但在柱状图中以浅蓝色展示，作为"最近发展区"的可视化。
```

注意：得分率与原积分是不同概念：
- 原积分 = 所有题目得分之和（0+1+2），用于查常模表
- 得分率 = P题数/总题数，用于判断薄弱领域和图表展示

柱状图中：
- 深蓝柱宽度 = P / 总题数（与得分率百分比完全对应）
- 浅蓝柱宽度 = E / 总题数（紧接深蓝柱后面，展示萌芽空间）
- 两段柱子加起来 = (P+E) / 总题数

### 6.3 颜色标注

```
得分率 ≤ 65%：领域名和百分比显示为红色（薄弱，需干预）
得分率 > 65%：领域名和百分比显示为绿色（达标）
```

### 6.4 右侧标注

```
每行右侧显示：
  {百分比}%
  通过{P数} 萌芽{E数} 不通过{F数}
```

### 6.5 图例

```
■ 通过(P)    ■ 萌芽(E)    --- 65% 薄弱线
```

## 8. 综合分析生成规则

综合分析为一段完整文本，融合整体概述、发展分析、行为分析和建议：

```
儿童当前实足年龄为X岁X个月，PEP-3 评估涵盖6个发展副测验和4个行为副测验。
发展副测验中，通过(P)共__题，萌芽(E)共__题，不通过(F)共__题，
整体发展年龄约__岁__月（沟通发展年龄__月，体能发展年龄__月）。

从各领域得分率看，相对优势领域为______（得分率___%）和______（得分率___%），
达到或接近同龄自闭症儿童的中等水平。
当前薄弱领域为______（得分率___%）和______（得分率___%），
得分率低于65%，建议作为重点干预方向。
萌芽技能集中在______领域（E=__题），提示该领域最近发展区较宽，干预潜力较大。

行为表现方面，______评级为S（严重），______评级为M（轻微），______评级为A（适当）。
行为合成分数百分比级数为__，发展/适应程度为______。
严重行为问题需优先处理，以减少对教学和日常生活的干扰。

照顾者报告显示：问题行为得分__/20，个人自理得分__/26，适应行为得分__/30。
照顾者评估的发展程度与测试员评估______（基本一致/存在差异，需关注）。

综合建议：
1. 优先干预______等严重行为问题
2. 重点加强______等薄弱发展领域，从萌芽技能切入
3. 关注______等轻度行为问题
4. 家庭中可增加亲子互动，鼓励孩子主动表达需求
```

**填充规则**：
- 优势领域 = 得分率 > 65% 的发展副测验，按得分率降序
- 薄弱领域 = 得分率 ≤ 65% 的发展副测验，按得分率升序
- 行为评级：A数/总题数 ≥ 75% → A，S数/总题数 ≥ 25% → S，其余 → M
- 建议按 P0~P4 优先级排列（见第6节）
- "______"必须用实际数据填充，不能留空

## 9. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- 原积分 = 该副测验所有题目得分之和（0+1+2），用于查常模表得到标准分和百分比级数
- 得分率 = P题数(得2分) / 总题数，用于判断薄弱领域和图表展示
- 发展数据分析图表中，≤65% 标红，>65% 标绿
- 薄弱线（65%红色虚线）在最上层，不被柱子遮挡
