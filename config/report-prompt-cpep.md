# CPEP-3 评估报告 — 计算规则

> 本文档定义 CPEP-3（自闭症儿童心理教育评核）评估报告的所有计算规则。
> CPEP-3 采用与 PEP-3 相同的标准结构（13个副测验 + 3个合成分数）。
> 同一份评估数据，必须产出完全相同的报告结果。

## 2. 评分体系

每题三级评分：
- **P（通过）= 2分**：独立完成
- **E（萌芽）= 1分**：部分完成或需要提示
- **F（不通过）= 0分**：无反应或无法完成

---

## 3. 副测验分数计算

### 3.1 原始分

```
原始分 = P题数（通过的题目数）
```

### 3.2 得分率

```
得分率 = P题数 / 总题数 × 100%
```

四舍五入取整。

### 3.3 发展年龄推算

基于得分率和儿童实足年龄（月数）：

| 得分率 | 发展年龄 |
|--------|---------|
| ≥ 90% | 实足年龄 |
| 80%~89% | 实足年龄 - 6个月 |
| 70%~79% | 实足年龄 - 12个月 |
| 60%~69% | 实足年龄 - 18个月 |
| 50%~59% | 实足年龄 - 24个月 |
| 40%~49% | 实足年龄 - 30个月 |
| < 40% | 实足年龄 - 36个月 |

发展年龄最低显示为 `<12`（月）。

注意：副测验 7~10（AE、SR、CMB、CVB）和照顾者报告 2~3（PB、AB）不适用发展年龄，显示为空。

### 3.4 百分比级数

| 得分率 | 百分比级数 |
|--------|-----------|
| ≥ 90% | 95 |
| 80%~89% | 75 |
| 70%~79% | 50 |
| 60%~69% | 25 |
| 50%~59% | 10 |
| < 50% | 按 得分率/10 四舍五入，最低为 1 |

### 3.5 发展/适应程度

| 百分比级数 | 程度 |
|-----------|------|
| ≥ 75 | 适当 |
| 50~74 | 轻度 |
| < 50 | 严重 |

## 4. 合成分数计算

### 4.1 标准分

```
各子测验标准分 = 该子测验的百分比级数（简化处理）
```

### 4.2 合成标准分总和

```
沟通(C) 标准分总和 = CVP百分比级数 + EL百分比级数 + RL百分比级数
体能(M) 标准分总和 = FM百分比级数 + GM百分比级数 + VMI百分比级数
行为(MB) 标准分总和 = AE百分比级数 + SR百分比级数 + CMB百分比级数 + CVB百分比级数
```

### 4.3 合成百分比级数

```
合成得分率 = 合成标准分总和 / (组成子测验数 × 95) × 100%
合成百分比级数 = 按 3.4 的映射规则
```

### 4.4 合成发展/适应程度

同 3.5 的映射规则。

### 4.5 合成发展年龄

同 3.3 的映射规则，基于合成得分率。

## 5. 图表

### 5.1 发展功能量表侧面图

按 CPEP-3 原始量表格式，7 个功能发展领域 + 总评分列 + 年龄列。

**列定义**

| 列 | 对应副测验 | 满分（题数） |
|----|----------|------------|
| 模仿 | VMI | 10 |
| 知觉 | CVP | 34 |
| 精细动作 | FM | 20 |
| 粗大动作 | GM | 18 |
| 手眼协调 | EL | 25 |
| 认知表现 | RL | 26 |
| 口语认知 | CVB | 11 |
| 总评分 | 以上7列之和 | 144 |

**坐标系**

- 左侧 Y 轴：年龄（1~6岁）
- 每列独立刻度：0 到该列满分，**每分一个刻度**
- 总评分列：0~144，每分一个刻度（每5分标数字）
- 右侧 Y 轴：年龄（1~6岁）
- 所有列共享同一个 globalMax（= 最大列满分 34），刻度按 `v / globalMax` 比例定位

**数据点**

```
● P 分数据点 = 该领域 P 题数（通过数）
  → Y 位置 = P / globalMax × chartHeight
  → 实线蓝色连接

△ P+E 分数据点 = 该领域 P 题数 + E 题数
  → Y 位置 = min(P+E, 该列满分) / globalMax × chartHeight
  → 虚线橙色连接

总评分列：
  ● 总 P = 所有领域 P 之和
  △ 总 P+E = 所有领域 (P+E) 之和
  → Y 位置按 totalMax(144) 比例定位
```

### 5.2 行为病理行为靶心图

5 个行为领域的病理行为严重程度可视化。

**领域定义**

| 领域 | 总题数 |
|------|--------|
| 情感 | 6 |
| 人际关系、合作行为及对人的兴趣 | 7 |
| 游戏及物品喜好 | 6 |
| 感觉模式 | 14 |
| 语言 | 11 |

**图形结构**

- 5 个扇形**等分**（每个 72°），从正上方（情感）开始顺时针排列
- 每个扇形内部的**环数 = 该领域总题数**（环数不同）
- 环宽 = maxR / 该扇形环数（题数少的环宽，题数多的环窄）

**S/M/A 统计**

每个领域的每道题评为 S（严重）、M（轻度）或 A（适当）三级之一：

```
S 数量 = 该领域中评为"严重"的题目数
M 数量 = 该领域中评为"轻度"的题目数
A 数量 = 该领域中评为"适当"的题目数
S + M + A = 该领域总题数
```

**填充规则**

```
从圆心向外：
  第 1 ~ S 环：涂黑（S 严重）
  第 S+1 ~ S+M 环：涂灰（M 轻度）
  第 S+M+1 ~ total 环：留白（A 适当）

等效比例：
  S 填充半径 = (S / total) × maxR
  M 填充半径 = ((S + M) / total) × maxR
  A 留白 = 剩余到 maxR
```

## 6. 综合分析生成规则

| 模块 | 内容规则 |
|------|---------|
| 概述 | 包含13个副测验名称和3个合成分数 |
| 优势领域 | 得分率 ≥ 70% 的副测验，按得分率降序 |
| 需关注领域 | 得分率 ≤ 40% 的副测验，按得分率升序 |
| 合成分数 | 列出沟通(C)、体能(M)、行为(MB)的标准分总和和发展适应程度 |

---

## 7. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
