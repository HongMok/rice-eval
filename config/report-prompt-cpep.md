# C-PEP-3 评估报告 — 计算规则

> 本文档定义 C-PEP-3（孤独症谱系及相关发育障碍儿童心理教育量表·中文修订三版）评估报告的所有计算规则。
> 同一份评估数据，必须产出完全相同的报告结果。

---

## 1. 评估结构

C-PEP-3 包含三大板块：

| 板块 | CSV一级分类 | 行数 | 评分体系 |

| 补充项目 | 补充项目 | 32 | 1=H(通过), 0=L(不通过) |

注：记分表和侧面图中，模仿(动作)+模仿(口语)合并为"模仿"，知觉(视觉)+知觉(听觉)合并为"知觉"，共7列。

### 1.2 病理行为量表（5个领域，44题）

| 领域 | CSV二级分类 | 题数 |

## 3. 记分表

### 3.1 表格结构

记分表分左右两部分并排展示：

**左侧：病理量表**

```
              情感  人际关系  物品喜好  感觉模式  语言
各项总和 #      6     7       6       14      11
合计数 A       {A}   {A}     {A}     {A}     {A}
合计数 M       {M}   {M}     {M}     {M}     {M}
合计数 S       {S}   {S}     {S}     {S}     {S}
```

**右侧：发展功能量表**

```
              模仿  知觉  精细动作  粗大动作  手眼协调  认知表现  口语认知
各项总和 #      10   11    10      11      14      20      20
P              {P}  {P}   {P}     {P}     {P}     {P}     {P}
E              {E}  {E}   {E}     {E}     {E}     {E}     {E}
F              {F}  {F}   {F}     {F}     {F}     {F}     {F}
```

**发展评分 = 所有发展领域P数之和（P总分）**

### 3.2 计算规则

```
每个发展领域：
  P数 = 得分为 2（通过）的题目数
  E数 = 得分为 1（萌芽）的题目数
  F数 = 得分为 0（不通过）的题目数
  P + E + F = 该领域总题数

每个病理行为领域：
  A数 = 得分为 2（恰当）的题目数
  M数 = 得分为 1（轻微）的题目数
  S数 = 得分为 0（严重）的题目数
  A + M + S = 该领域总题数

发展评分（P总分）= 所有7个发展领域的P数之和
```

## 4. 发展功能量表侧面图

### 4.1 图表结构

- X轴 = 7个发展领域 + 总评分列 + 年龄列
- Y轴 = 年龄（岁），范围 1~7岁
- 背景 = 常模表数值（粉色条纹行）

7个领域列顺序：模仿、知觉、精细动作、粗大动作、手眼协调、认知表现、口语认知

### 4.2 两条数据线

**● P 分（圆点实线连接）**

```
每个领域的 P 值 = 该领域中得分为 2（通过）的题目数
总评分列的 P 值 = 所有领域 P 之和（= 发展评分）
在常模表中找到 P 值对应的年龄行 → 画圆点 → 实线连接
```

**△ P+E 分（三角形虚线连接）**

```
每个领域的 P+E 值 = P数 + E数
总评分列的 P+E 值 = 所有领域 (P+E) 之和
在常模表中找到 P+E 值对应的年龄行 → 画三角形 → 虚线连接
```

### 4.3 P+E 与 P 的差距 = 萌芽技能（最近发展区）

```
差距越大 → 该领域的最近发展区越宽 → 干预潜力越大
```

### 4.4 总评分→发展年龄

```
总评分列的 P 值 → 在常模表右侧年龄列查找 → 得到总体发展年龄
```

### 4.5 底部汇总

```
通过项目分数：{各领域P值} （总和）
部分通过项目分数：{各领域P+E值} （总和）
```

## 5. 病理行为量表靶心图

### 5.1 图形结构

- 5 个扇形等分（每个 72°），从正上方开始顺时针排列
- 顺序：情感 → 人际关系 → 游戏及物品喜好 → 感觉模式 → 语言
- 每个扇形内部的环数 = 该领域总题数（环数不同）

### 5.2 填充规则

```
从圆心向外：
  第 1 ~ S 环：涂黑/涂红（S 严重项目数）
  第 S+1 ~ S+M 环：涂灰（M 轻微项目数）
  第 S+M+1 ~ total 环：留白（A 恰当项目数）
```

### 5.3 图例

```
■ 严重项目数（黑色/红色）    ■ 轻度项目数（灰色）
```

### 5.4 说明文字

```
[说明] 根据每个病理行为领域中评为严重（S）的项目数，从圆心开始涂黑相应的环数。
再根据各病理行为领域中评为轻度（M）的项目数，继续将相应的环数涂上阴影。
其余的留作空白。
```

## 6. 得分率计算（用于综合分析和IEP）

### 6.1 发展量表得分率

```
领域得分率 = P数 / 总题数 × 100%

只计算完全通过(P)的题目占比。
萌芽(E)不计入得分率，但在侧面图中以△P+E线展示最近发展区。
```

### 6.2 领域评价

| 得分率 | 评价 |
|--------|------|
| ≥ 75% | 强处 🟢 |
| ≤ 40% | 弱处 🔴 |
| 其余 | 不标注 |

### 6.3 病理行为评级

```
领域严重率 = S数 / 总题数 × 100%
```

| 严重率 | 评级 |
|--------|------|
| ≥ 50% | S（严重） |
| 20%~49% | M（轻度） |
| < 20% | A（适当） |

## 7. 综合分析生成规则

综合分析为一段完整文本，融合发展年龄对比、发展分析、行为分析、补充项目和建议：

```
{姓名}生理年龄为{X}岁{Y}月，本次C-PEP评估总体发展年龄约为{A}岁{B}月，
落后生理年龄约{C}个月。功能发展96题中，通过(P)共__题，萌芽(E)共__题，
不通过(F)共__题。

各领域发展年龄（按差距从大到小）：
- {领域}：发展年龄约{A}岁{B}月，落后约{C}个月
- ...

从各领域得分率看，相对优势领域为______（得分率___%，强处🟢），
当前薄弱领域为______（得分率___%，弱处🔴）和______（得分率___%）。
萌芽技能集中在______领域（E=__题），提示该领域最近发展区较宽，干预潜力较大。
萌芽项目明细：______。

病理行为方面，______评级为S（严重率≥50%），______评级为M，______评级为A。
严重异常项目：______。需优先干预严重行为问题，减少对教学的干扰。

补充项目评估发现：
- 能力上限：______等(H)项通过，说明有进一步发展潜力
- 基础能力：______等(L)项通过，具备基础能力
- 需重点关注：______等(L)项未通过，基础技能薄弱

综合建议（按优先级）：
1. 优先干预S级行为领域
2. 重点加强弱处领域中萌芽技能多的方向（最近发展区，干预效果最好）
3. 补充(L)未通过的基础技能训练
4. 关注M级行为领域
5. 家庭中增加亲子互动，鼓励主动表达
```

**填充规则**：
- 优势领域 = 得分率 ≥ 75% 的领域（强处🟢）
- 薄弱领域 = 得分率 ≤ 40% 的领域（弱处🔴）
- 发展年龄差距判定：≤6月=基本持平，7-12月=轻度落后，13-24月=中度落后，>24月=显著落后
- 病理行为评级：严重率≥50%→S，20%-49%→M，<20%→A
- 建议按 P1~P5 优先级排列
- "______"必须用实际数据填充，不能留空

## 8. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- 补充项目不计入发展量表和病理行为量表的总分
- 发展评分（P总分）= 只计得2分（通过）的题目数之和
- 得分率 = P数/总题数，萌芽(E)不计入得分率
