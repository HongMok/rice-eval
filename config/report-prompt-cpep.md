# C-PEP-3 评估报告 — 计算规则

> 本文档定义 C-PEP-3（孤独症谱系及相关发育障碍儿童心理教育量表·中文修订三版）评估报告的所有计算规则。
> 同一份评估数据，必须产出完全相同的报告结果。

---

## 1. 评估结构

C-PEP-3 包含三大板块：

| 板块 | CSV一级分类 | 行数 | 评分体系 |
|------|-----------|------|---------|
| 发展量表 | 发展 | 96 | 2=P(通过), 1=E(萌芽), 0=F(不通过) |
| 病理行为量表 | 病理学 | 44 | 2=A(恰当), 1=M(轻微), 0=S(严重) |
| 补充项目 | 补充项目 | 32 | 1=H(通过), 0=L(不通过) |

三大板块通过CSV一级分类直接区分：发展、病理学、补充项目。

### 1.1 发展量表（9个领域，96题）

| 领域 | CSV二级分类 | 题数 |
|------|-----------|------|
| 模仿 | 模仿(动作) + 模仿(口语) | 10 |
| 知觉 | 知觉(视觉) + 知觉(听觉) | 11 |
| 精细动作 | 精细动作 | 10 |
| 粗大动作 | 粗大动作 | 11 |

| 认知表现 | 认知表现 | 20 |
| 口语认知 | 口语认知 | 20 |

注：记分表和侧面图中，模仿(动作)+模仿(口语)合并为"模仿"，知觉(视觉)+知觉(听觉)合并为"知觉"，共7列。

### 1.2 病理行为量表（5个领域，44题）

| 领域 | CSV二级分类 | 题数 |
|------|-----------|------|
| 情感 | 情感 | 6 |
| 人际关系 | 人际关系、合作行为及对人的兴趣 | 7 |

补充项目不计入发展量表和病理行为量表的总分。
- **(H)** = 高阶扩展（主项目已通过时使用）

## 3. 记分表

### 3.1 表格结构

记分表分左右两部分并排展示：

**左侧：病理量表**

```
              情感  人际关系  物品喜好  感觉模式  语言
各项总和 #      6     7       6       14      11
合计数 A       {A}   {A}     {A}     {A}     {A}
合计数 M       {M}   {M}     {M}     {M}     {M}
合计数 S       {S}   {S}     {S}     {S}     {S}
```

**右侧：发展功能量表**

```
              模仿  知觉  精细动作  粗大动作  手眼协调  认知表现  口语认知
各项总和 #      10   11    10      11      14      20      20
P              {P}  {P}   {P}     {P}     {P}     {P}     {P}
E              {E}  {E}   {E}     {E}     {E}     {E}     {E}
F              {F}  {F}   {F}     {F}     {F}     {F}     {F}
```

**发展评分 = 所有发展领域P数之和（P总分）**

### 3.2 计算规则

```
每个发展领域：
  P数 = 得分为 2（通过）的题目数
  E数 = 得分为 1（萌芽）的题目数
  F数 = 得分为 0（不通过）的题目数
  P + E + F = 该领域总题数

每个病理行为领域：
  A数 = 得分为 2（恰当）的题目数
  M数 = 得分为 1（轻微）的题目数
  S数 = 得分为 0（严重）的题目数
  A + M + S = 该领域总题数

发展评分（P总分）= 所有7个发展领域的P数之和
```

## 4. 发展功能量表侧面图

### 4.1 图表结构

- X轴 = 7个发展领域 + 总评分列 + 年龄列
- Y轴 = 年龄（岁），范围 1~7岁
- 背景 = 常模表数值（粉色条纹行）

7个领域列顺序：模仿、知觉、精细动作、粗大动作、手眼协调、认知表现、口语认知

### 4.2 两条数据线

**● P 分（圆点实线连接）**

```
每个领域的 P 值 = 该领域中得分为 2（通过）的题目数
总评分列的 P 值 = 所有领域 P 之和（= 发展评分）
在常模表中找到 P 值对应的年龄行 → 画圆点 → 实线连接
```

**△ P+E 分（三角形虚线连接）**

```
每个领域的 P+E 值 = P数 + E数
总评分列的 P+E 值 = 所有领域 (P+E) 之和
在常模表中找到 P+E 值对应的年龄行 → 画三角形 → 虚线连接
```

### 4.3 P+E 与 P 的差距 = 萌芽技能（最近发展区）

```
差距越大 → 该领域的最近发展区越宽 → 干预潜力越大
```

### 4.4 总评分→发展年龄

```
总评分列的 P 值 → 在常模表右侧年龄列查找 → 得到总体发展年龄
```

### 4.5 底部汇总

```
通过项目分数：{各领域P值} （总和）
部分通过项目分数：{各领域P+E值} （总和）
```

## 5. 病理行为量表靶心图

### 5.1 图形结构

- 5 个扇形等分（每个 72°），从正上方开始顺时针排列
- 顺序：情感 → 人际关系 → 游戏及物品喜好 → 感觉模式 → 语言
- 每个扇形内部的环数 = 该领域总题数（环数不同）

### 5.2 填充规则

```
从圆心向外：
  第 1 ~ S 环：涂黑/涂红（S 严重项目数）
  第 S+1 ~ S+M 环：涂灰（M 轻微项目数）
  第 S+M+1 ~ total 环：留白（A 恰当项目数）
```

### 5.3 图例

```
■ 严重项目数（黑色/红色）    ■ 轻度项目数（灰色）
```

### 5.4 说明文字

```
[说明] 根据每个病理行为领域中评为严重（S）的项目数，从圆心开始涂黑相应的环数。
再根据各病理行为领域中评为轻度（M）的项目数，继续将相应的环数涂上阴影。
其余的留作空白。
```

## 6. 得分率计算（用于综合分析和IEP）

### 6.1 发展量表得分率

```
领域得分率 = P数 / 总题数 × 100%

只计算完全通过(P)的题目占比。
萌芽(E)不计入得分率，但在侧面图中以△P+E线展示最近发展区。
```

### 6.2 领域评价

| 得分率 | 评价 |
|--------|------|
| ≥ 75% | 强处 🟢 |
| ≤ 40% | 弱处 🔴 |
| 其余 | 不标注 |

### 6.3 病理行为评级

```
领域严重率 = S数 / 总题数 × 100%
```

| 严重率 | 评级 |
|--------|------|
| ≥ 50% | S（严重） |
| 20%~49% | M（轻度） |
| < 20% | A（适当） |

## 7. 综合分析生成规则

> 注意：以下为综合分析的内容结构，不要修改此结构。

### 7.1 发展年龄与生理年龄对比（必须放在综合分析最前面）

**总体发展年龄**

```
总体发展年龄 = 发展评分(P总分) → 查常模表 → 对应年龄（月）
简化公式：总体发展年龄（月）= (P总分 / 96) × 儿童实足年龄（月）

生理年龄 = 评估日期 - 出生日期（月）
发展差距 = 生理年龄 - 总体发展年龄（月）

格式：
{姓名}生理年龄为{X}岁{Y}月，本次评估总体发展年龄约为{A}岁{B}月，
落后生理年龄约{C}岁{D}月（或"与生理年龄基本持平"）。
```

**各领域发展年龄明细**

对每个发展量表领域，计算领域发展年龄和与生理年龄的差距：

```
领域发展年龄（月）= (领域P数 / 领域总题数) × 儿童实足年龄（月）
领域差距 = 生理年龄 - 领域发展年龄（月）
```

| 领域 | 总题数 | P数 | 得分率 | 发展年龄 | 与生理年龄差距 |
|------|--------|-----|--------|---------|-----------|
| 模仿 | 10 | {P} | {P/10}% | {发展年龄} | {差距} |
| 知觉 | 11 | {P} | {P/11}% | {发展年龄} | {差距} |
| 精细动作 | 10 | {P} | {P/10}% | {发展年龄} | {差距} |
| 粗大动作 | 11 | {P} | {P/11}% | {发展年龄} | {差距} |
| 手眼协调 | 14 | {P} | {P/14}% | {发展年龄} | {差距} |
| 认知表现 | 20 | {P} | {P/20}% | {发展年龄} | {差距} |
| 口语认知 | 20 | {P} | {P/20}% | {发展年龄} | {差距} |

输出格式：
```
各领域发展年龄：
- {领域}：发展年龄约{A}岁{B}月，落后生理年龄约{C}个月
- ...（按差距从大到小排列）
```

差距判定：
- 差距 ≤ 6个月：与生理年龄基本持平
- 差距 7~12个月：轻度落后
- 差距 13~24个月：中度落后
- 差距 > 24个月：显著落后

```
格式：
本次评估涵盖功能发展{7}个领域和行为表现{5}个领域。
功能发展总分{96}题中，已有能力(P){P总数}题，部分能力(E){E总数}题，
```

### 7.3 发展量表各领域分析

按得分率从低到高排列，对每个领域输出：

```
{领域名}：P={P数} E={E数} F={F数}，得分率 {Y}%
  评价：{强处/弱处/不标注}
  萌芽技能（E）：{列出评为E的具体项目名}
```

**萌芽技能（E）是报告的重点**——代表最近发展区，是干预的优先切入点。

### 7.4 病理行为分析

对每个行为领域输出：

```
{领域名}：A={A数} M={M数} S={S数}，严重率 {Y}%，评级 {A/M/S}
  异常项目：{列出评为S的具体项目名}
```

### 7.5 补充项目分析

```
补充项目评估发现：
- 能力上限：{列出(H)项通过的项目} → 说明在{领域}有进一步发展潜力
- 基础能力：{列出(L)项通过的项目} → 说明具备{领域}的基础能力
- 需重点关注：{列出(L)项未通过的项目} → 说明{领域}基础技能薄弱
```

### 7.6 建议

```
优先级：
P1：S级行为领域（严重行为问题优先处理）
P2：发展量表弱处领域中萌芽技能(E)多的（最近发展区，干预效果最好）
P3：补充项目中(L)未通过的领域（基础技能缺失）
P4：M级行为领域
P5：发展量表弱处领域中萌芽技能(E)少的
```

## 8. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- 补充项目不计入发展量表和病理行为量表的总分
- 发展评分（P总分）= 只计得2分（通过）的题目数之和
- 得分率 = P数/总题数，萌芽(E)不计入得分率
