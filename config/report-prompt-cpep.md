> 同一份评估数据，必须产出完全相同的报告结果。

## 2. 发展量表得分计算

### 2.1 领域得分

```
领域P数 = 该领域中评分为 P(2分) 的题目数
领域E数 = 该领域中评分为 E(1分) 的题目数
领域F数 = 该领域中评分为 F(0分) 的题目数
领域总题数 = P + E + F
领域得分 = P × 2 + E × 1 + F × 0
领域满分 = 总题数 × 2
领域得分率 = 领域得分 / 领域满分 × 100%
```

### 2.2 发展量表总分

```
发展总分 = 所有发展领域得分之和
发展满分 = 96 × 2 = 192
发展总得分率 = 发展总分 / 192 × 100%
```

### 2.3 领域评价

| 得分率 | 评价 |
|--------|------|
| ≥ 75% | 强处 🟢 |
| ≤ 40% | 弱处 🔴 |
| 其余 | 不标注 |

## 3. 病理行为量表得分计算

### 3.1 A/M/S 统计

对每个行为领域，统计三种评级的题目数：

```
A数量 = 该领域中评为 A(恰当，2分) 的题目数
M数量 = 该领域中评为 M(轻微，1分) 的题目数
S数量 = 该领域中评为 S(严重，0分) 的题目数
```

### 3.2 行为领域评级

```
领域严重率 = S / 总题数 × 100%
```

| 严重率 | 评级 | 含义 |
|--------|------|------|
| ≥ 50% | S（严重） | 该领域行为异常严重 |
| 20%~49% | M（轻度） | 该领域存在轻度异常 |
| < 20% | A（适当） | 该领域行为基本正常 |

## 4. 补充项目分析

### 4.1 补充项目得分

```
领域H数 = 该领域中评为 H(通过，1分) 的题目数
领域L数 = 该领域中评为 L(不通过，0分) 的题目数
领域通过率 = H / (H + L) × 100%
```

### 4.2 补充项目在报告中的作用

补充项目**不计入**发展量表总分和病理行为量表总分，但在综合分析中提供以下信息：

1. **能力上限探测**：(H)项通过 → 说明该技能领域有进一步发展的潜力
2. **能力下限探测**：(L)项通过 → 说明虽然主项目未通过，但具备基础能力
3. **最近发展区定位**：(H)项未通过 → 标记为当前能力天花板；(L)项未通过 → 标记为需要重点干预的基础技能

### 4.3 报告输出格式

```
补充项目评估结果：
- {二级分类}：{H数}/{总数} 通过
  - {评估项目名}：{H/L}（{操作描述摘要}）
  - ...
```

仅列出有意义的发现（如(H)通过或(L)未通过的项目），不逐条罗列全部补充项目。

## 5. 发展功能侧面图

### 5.1 图表结构

横轴 = 9个发展领域 + 总评分列
纵轴 = 得分

### 5.2 两条数据线

**● P 分（实线）**

```
每个领域的 P 值 = 该领域 P(通过) 的题目数
总评分列的 P 值 = 所有领域 P 之和
```

**△ P+E 分（虚线）**

```
每个领域的 P+E 值 = P数 + E数
总评分列的 P+E 值 = 所有领域 (P+E) 之和
```

P+E 线与 P 线之间的差距 = 萌芽技能数量，代表**最近发展区**。

## 7. 综合分析生成规则

### 7.1 发展年龄与生理年龄对比（必须放在综合分析最前面）

**总体发展年龄**

```
总体发展年龄 = 发展量表总得分(P×2+E×1) → 查常模表 → 对应年龄（月）
简化公式：总体发展年龄（月）= (发展量表总得分 / 192) × 儿童实足年龄（月）

生理年龄 = 评估日期 - 出生日期（月）
发展差距 = 生理年龄 - 总体发展年龄（月）

格式：
{姓名}生理年龄为{X}岁{Y}月，本次评估总体发展年龄约为{A}岁{B}月，
落后生理年龄约{C}岁{D}月（或"与生理年龄基本持平"）。
```

**各领域发展年龄明细**

对每个发展量表领域，计算领域发展年龄和与生理年龄的差距：

```
领域发展年龄（月）= (领域得分 / 领域满分) × 儿童实足年龄（月）
领域差距 = 生理年龄 - 领域发展年龄（月）
```

| 领域 | 总题数 | 满分 | 得分 | 得分率 | 发展年龄 | 与生理年龄差距 |
|------|--------|------|------|--------|---------|-----------|
| 精细动作 | 10 | 20 | {得分} | {得分率}% | {发展年龄} | {差距} |
| 模仿(动作) | 5 | 10 | {得分} | {得分率}% | {发展年龄} | {差距} |
| 模仿(口语) | 5 | 10 | {得分} | {得分率}% | {发展年龄} | {差距} |
| 知觉(视觉) | 8 | 16 | {得分} | {得分率}% | {发展年龄} | {差距} |
| 知觉(听觉) | 3 | 6 | {得分} | {得分率}% | {发展年龄} | {差距} |
| 认知表现 | 20 | 40 | {得分} | {得分率}% | {发展年龄} | {差距} |
| 口语认知 | 20 | 40 | {得分} | {得分率}% | {发展年龄} | {差距} |
| 手眼协调 | 14 | 28 | {得分} | {得分率}% | {发展年龄} | {差距} |
| 粗大动作 | 11 | 22 | {得分} | {得分率}% | {发展年龄} | {差距} |

输出格式：
```
各领域发展年龄：
- 感知觉：发展年龄约{A}岁{B}月，落后生理年龄约{C}个月
- 粗大动作：发展年龄约{A}岁{B}月，落后生理年龄约{C}个月
- ...（按差距从大到小排列）
```

差距判定：
- 差距 ≤ 6个月：与生理年龄基本持平
- 差距 7~12个月：轻度落后
- 差距 13~24个月：中度落后
- 差距 > 24个月：显著落后

### 7.6 建议

```
优先级：
P1：S级行为领域（严重行为问题优先处理）
P2：发展量表弱处领域中萌芽技能(E)多的（最近发展区，干预效果最好）
P3：补充项目中(L)未通过的领域（基础技能缺失）
P4：M级行为领域
P5：发展量表弱处领域中萌芽技能(E)少的
```

## 8. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- 补充项目不计入发展量表和病理行为量表的总分
