# C-PEP 评估报告 — 计算规则

> 本文档定义 C-PEP（孤独症儿童发展评估表）评估报告的所有计算规则。
> 同一份评估数据，必须产出完全相同的报告结果。

---

情绪与行为（CSV一级分类="情绪与行为"）共 52 题，按 5 个行为领域分组：

| 行为领域 | 报告显示名 | 包含的CSV二级分类 | 题数 | 评分体系 |
|---------|---------|----------------|------|---------|
| 情感 | 情感 | 回应行为反应、依恋情绪行为、情绪识别、回应他人、表达情绪、调节情绪、关系与情感、接纳亲近 | 16 |  没有/轻度/严重 |

## 2. 功能发展量表得分计算

### 2.1 领域得分

```
领域通过数(P) = 该领域中得分为 1（通过）的题目数
领域总题数 = 该领域的全部题目数
领域得分率 = P / 总题数 × 100%
```

### 2.2 各领域满分表

| 领域 | 总题数 | 满分（=总题数） |
|------|--------|------------|
| 感知觉 | 55 | 55 |
| 粗大动作 | 72 | 72 |
| 精细动作 | 66 | 66 |
| 语言与沟通 | 79 | 79 |
| 认知 | 55 | 55 |
| 社会交往 | 47 | 47 |
| 生活自理 | 67 | 67 |
| **合计** | **441** | **441** |

### 2.3 功能发展总分

```
功能发展总分 = 所有 7 个领域的通过数之和
满分 = 441
```

### 2.4 发展年龄

通过领域通过数查**发展功能量表侧面图常模表**得到发展年龄（岁）。

查表方法同 PEP-3：在常模表的该领域列中，找到通过数对应的年龄行。

**简化映射规则**（当无法获取完整常模表时使用）：

```
发展年龄（月）= 得分率 × 儿童实足年龄（月）
发展年龄最低显示为 <12（月）
```

### 2.5 领域评价

| 得分率 | 评价 |
|--------|------|
| ≥ 75% | 强处 🟢 |
| ≤ 40% | 弱处 🔴 |
| 其余 | 不标注 |

## 3. 病理行为量表得分计算

### 3.1 S/M/A 统计

对每个行为领域，统计三种评级的题目数：

```
S 数量 = 该领域中评为"严重"的题目数
M 数量 = 该领域中评为"轻度"的题目数
A 数量 = 该领域中评为"没有"的题目数
S + M + A = 该领域总题数
```

### 3.2 行为领域评级

```
领域严重率 = S / 总题数 × 100%
```

| 严重率 | 评级 | 含义 |
|--------|------|------|
| ≥ 50% | S（严重） | 该领域行为异常严重 |
| 20%~49% | M（轻度） | 该领域存在轻度异常 |
| < 20% | A（适当） | 该领域行为基本正常 |

### 4.1 图表结构

横轴 7 个功能发展领域 + 总评分列 + 年龄列：

| 列 | 对应领域 | 满分 |
|----|---------|------|

| 粗大动作 | 粗大动作 | 72 |
| 精细动作 | 精细动作 | 66 |
| 语言与沟通 | 语言与沟通 | 79 |

| 社会交往 | 社会交往 | 47 |
| 生活自理 | 生活自理 | 67 |
| 总评分 | 以上7列之和 | 441 |

纵轴 = 年龄（岁），范围 1~7 岁。

### 4.2 两条数据线

**● P 分（实线连接）**

```
每个领域的 P 值 = 该领域通过数（得分为1的题目数）
总评分列的 P 值 = 所有领域 P 之和
```

**△ P+E 分（虚线连接）**

C-PEP 是 2 级评分（通过/不通过），没有"萌芽(E)"。但如果使用 3 级评分（0=不通过, 0.5=中间反应, 1=通过），则：

```
每个领域的 P+E 值 = 该领域通过数 + 中间反应数
总评分列的 P+E 值 = 所有领域 (P+E) 之和
```

如果是 2 级评分，P+E 线与 P 线重合。

### 4.3 年龄对应

```
总评分的 P 值 → 查常模表右侧年龄列 → 得到发展年龄
```

## 5. 病理行为靶心图

### 5.1 图形结构

- 5 个扇形等分（每个 72°），从正上方（情感）开始顺时针排列
- 顺序：情感 → 人际关系 → 游戏及物品喜好 → 感觉模式 → 语言
- 每个扇形内部的环数 = 该领域总题数（环数不同）

### 5.2 填充规则

```
从圆心向外：
  第 1 ~ S 环：涂黑/涂红（S 严重）
  第 S+1 ~ S+M 环：涂灰/斜线（M 轻度）
  第 S+M+1 ~ total 环：留白（A 适当）
```

### 5.3 计算示例

```
情感领域：总题数 16
  S=2, M=3, A=11
  涂黑 2 环 → 涂灰 3 环 → 留白 11 环

感觉模式领域：总题数 10
  S=0, M=1, A=9
  涂黑 0 环 → 涂灰 1 环 → 留白 9 环
```

## 6. 综合分析生成规则

### 6.1 各领域得分概述

```
格式：本次评估涵盖功能发展{7}个领域和行为表现{5}个领域。
功能发展总分{441}分中，通过(P){P总数}分。整体发展年龄约{X}岁{Y}月。
```

### 6.2 优势领域

```
列出所有得分率 ≥ 60% 的功能发展领域，按得分率降序
格式：{领域名}({得分率}%)
```

### 6.3 需关注领域

```
列出所有得分率 ≤ 40% 的功能发展领域，按得分率升序
格式：{领域名}({得分率}%)，得分率偏低，建议作为重点干预领域。
```

### 6.4 行为表现

```
列出所有行为领域的评级
格式：{领域名}({评级}-{含义})
```

### 6.5 建议

```
优先级：P1=S级行为领域 → P2=弱处功能领域 → P3=M级行为领域
```

## 7. 确定性要求

- 同一份输入数据 → 同一份报告输出，无例外
- 不使用随机措辞、不使用"可能""大概"等模糊表述
- 所有数值计算使用四舍五入
- 所有阈值判断使用闭区间（含边界值）
- C-PEP 功能发展量表为 2 级评分（通过/不通过），满分=总题数
