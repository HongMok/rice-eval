### 功能说明

**整体布局**
- 顶部栏：返回 + 工具名称 + 进度条 + 设置按钮
- 左侧导航树（240px）：分组→领域两级结构
- 右侧题目卡片区域：当前领域的所有题目
- 底部操作栏：待定信息 + 快捷操作 + 提交按钮

**左侧导航树**
- 分组标题可折叠（`navCollapsedByDefault` 控制默认状态）
- 领域项显示圆点状态：灰色=未答、橙色=部分完成、绿色=全部完成
- 领域项右侧显示进度数字 `(已答/总数)`

**题目卡片 — 直接评分模式**
- 卡片头部：题号 + 项目名称 + 待定标记按钮（🚩）
- 信息盒子：操作描述、所需材料、适用年龄（由 `cardFields` 配置控制）
- 操作描述支持换行显示（CSV 中的 `\n` 在导入时转为真正换行符）
- 评分按钮组：过滤掉分值为空的等级，按分值从低到高排列（左→右），颜色红→橙→绿
- 评分按钮显示格式：`X分（标签）`，如 `1分（达标）`、`0.5分（4个）`；无标签时只显示 `X分`
- 评分描述：仅 `showScoreDesc=true` 时显示（C-PEP 启用）

**题目卡片 — 回合统计模式（Cold Probe）**
- 评分按钮区域替换为回合统计区域
- 默认 3 个回合行，每行 3 个按钮：独立(+)绿色 / 错误(-)红色 / 辅助(P)橙色
- 可点击"+ 新增测试回合"增加回合，无上限
- 回合统计完成后自动显示换算结果

**回合统计换算公式**

用独立率作为百分比，线性映射到该题的分值区间，四舍五入到最近的有效评分等级。天然兼容3级、4级、5级评分体系。

```
独立率 = 独立完成数 / 已答回合数
辅助率 = 辅助完成数 / 已答回合数

// Step 1: 计算有效比率 effectiveRate（0~1）
if (独立率 >= 80%)        → effectiveRate = 1.0（稳定掌握）
else if (全部错误)         → effectiveRate = 0（完全未掌握）
else if (独立=0 且 辅助>0) → effectiveRate = 1/(等级数-1)（有辅助能力，略高于最低分）
else                       → effectiveRate = 独立率（线性映射）

// Step 2: 线性插值到分值
rawValue = 最低分 + effectiveRate × (最高分 - 最低分)

// Step 3: 四舍五入到最近的有效评分等级
resultScore = 距离 rawValue 最近的评分等级
```

**辅助完成的特殊处理**

辅助完成 (P) 不计入独立完成，但不等于错误：
- **全辅助无独立**：effectiveRate = `1/(等级数-1)`，映射到略高于最低分的位置（第2个等级附近）
  - 3级 → effectiveRate=0.5 → 中间分
  - 4级 → effectiveRate=0.33 → 第2等级
  - 5级 → effectiveRate=0.25 → 第2等级
- **全错误无辅助**：effectiveRate = 0 → 最低分
- 设计意图：有辅助下能完成说明技能处于萌芽状态，应高于完全不会

**模式切换规则**

| 切换方向 | 数据处理 | 确认提示 |
|---------|---------|---------|
| 直接评分 → 回合统计 | 评分数据保留，回合数据为空 | "切换后需通过回合统计重新验证" |
| 回合统计 → 直接评分 | 评分数据保留，回合数据清除 | "回合统计数据将被清除" |

- 模式为全局设置（影响所有题目）
- 默认模式：`rounds`（回合统计）

**提交校验规则**

| 条件 | 行为 |
|------|------|
| 0 题已答 | 阻断提交，提示"请至少完成部分题目" |
| 部分已答 + `allowPartialSubmit=false` | 阻断提交，提示未完成题数 |
| 部分已答 + `allowPartialSubmit=true` | 确认弹窗，显示已完成/未完成题数 |
| 全部完成 | 确认弹窗，确认后提交 |

- PEP-3 / C-PEP / VB-MAPP 均为 `allowPartialSubmit=false`
- 提交后：更新记录 `status=completed` → 跳转结果路由页

**底部操作栏**
- 左侧：待定计数 + 查看链接（弹出待定列表弹窗）
- 中间：快捷操作按钮（一键评最低分 / 一键评最高分 / 下一领域）
- 右侧：提交按钮（未全部完成=灰色，全部完成=蓝色）

**备注功能**
- 折叠式，点击"📝 备注"展开输入框
- 200 字限制，失焦自动保存

**待定标记**
- 点击🚩切换标记状态，被标记卡片左侧显示橙色竖条
- 待定列表弹窗：按领域分组展示，点击可跳转定位

**断点续答**
- 每次评分操作后立即自动保存
- 退出时弹出确认弹窗，提示"进度已自动保存"
- 恢复时加载所有已保存数据（scores、notes、marks、roundsData）
