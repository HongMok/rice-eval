### 功能说明

**三色主题设计**

VB-MAPP 报告采用三色主题区分三大评估板块：

| 板块 | 主题色 | 渐变色 | 说明 |
|------|--------|--------|------|
| 里程碑评估 | 🟢 绿色 | #43A047 → #66BB6A | 发展性技能评估 |
| 障碍评估 | 🟠 橙色 | #F9A825 → #FFB74D | 学习障碍评估 |
| 转衔评估 | 🔵 蓝色 | #1E88E5 → #42A5F5 | 转衔准备评估 |

每个板块包含：渐变色标题栏 + 计分总表 + 评估报告（领域卡片列表）

**Tab1 — 评估报告内容模块**

| 模块 | 说明 |
|------|------|
| 基本信息 | 姓名、性别、出生日期、测评日期 |
| 里程碑计分总表 | 得分概览条 + 三级网格（Level 3/2/1），每级按领域展示填充格 |
| 里程碑评估报告 | 16 个领域卡片，每卡片含：领域名+得分、评价描述、IEP建议 |
| 障碍计分总表 | 得分概览条 + 24项障碍网格（每项4分） |
| 障碍评估报告 | 24 项障碍卡片，含得分、影响描述、干预建议 |
| 转衔计分总表 | 得分概览条 + 按三大类别分组的网格（每项5分） |
| 转衔评估报告 | 18 个转衔领域卡片，按三大类别分组展示 |
| 综合分析 | 自动生成，支持编辑保存 |

**里程碑评估领域（16个）**

```
提要求、命名、听者反应、视觉配对、游戏、社交、模仿、仿说、
发音、LRFFC、对话、集体技能、语言结构、数学、阅读、书写
```

**里程碑三级结构**

| 阶段 | 月龄范围 | 说明 |
|------|---------|------|
| Level 1 | 0-18月 | 基础阶段 |
| Level 2 | 18-30月 | 中级阶段 |
| Level 3 | 30-48月 | 高级阶段 |

**障碍评估项目（24项）**

```
行为问题、教学控制、提要求的缺陷、命名的缺陷、模仿的缺陷、
仿说的缺陷、视觉感知和配对的缺失、听者技能的缺陷、对话的缺陷、
社会技能的缺陷、依赖辅助、猜想式回答、扫视的缺陷、
条件性辨别的缺陷、不能泛化、薄弱动机、对行为有要求就会减弱动机、
依赖强化物、自我刺激、发音清晰度的缺陷、强迫性行为、
多动行为、没有目光接触、感觉防御
```

每项满分4分，得分越高表示障碍越严重。

**转衔评估三大类别（18个领域）**

| 类别 | 包含领域 |
|------|---------|
| 第一类：VB-MAPP得分和独立学习的能力 | VB-MAPP里程碑得分、VB-MAPP障碍得分、负面行为和教学控制、教室常规、集体技能、社会技能和社会游戏、独立的学业工作 |
| 第二类：学习模式 | 泛化、强化物泛化、获得技能的速度、新技能的维持、自然环境中的学习、未经训练的转换 |
| 第三类：语言和社交技能 | 自发语言、独立的对话、社交互动、同伴互动、社区活动 |

每项满分5分。

> 报告计算规则详见：`config/report-prompt-vbmapp.md`

**Tab2 — IEP**
- 详见界面9：IEP 个别化教育计划
- IEP 生成规则详见：`config/iep-prompt-vbmapp.md`

**Tab3 — 评估量表**
- 详见界面10：评估量表详情
- 左侧导航：按里程碑三级阶段分组（Level 1/2/3），每级下列出各领域
- 评分选项：达标(1) / 部分达标(0.5) / 未达标(0)

**底部操作栏**
- 【导出PDF】：调用 `window.print()`，打印样式适配三色主题
