### 功能说明

**基本信息区**
- 学生姓名、性别、出生日期、使用日期（可选择）、制定人员、家长签名

**课程包选择器**
- 下拉选择课程包（数据来源：`config/course-packets.json`）
- 选择后根据评估结果自动生成 IEP 建议
- 未选择时显示提示："请先选择课程包，系统将根据评估结果自动生成 IEP 建议"

**IEP 表格**
- 三列：年龄段及领域、干预目标、阶段目标
- 自动生成逻辑：筛选薄弱领域 → 匹配课程包内干预项目 → 填充阶段目标

> IEP 生成规则详见：
> - PEP-3：`config/iep-prompt-pep3.md`
> - C-PEP：`config/iep-prompt-cpep.md`
> - VB-MAPP：`config/iep-prompt-vbmapp.md`
> - 通用工具：`config/iep-prompt-common.md`

**编辑模式**
- 点击"编辑"进入编辑态，表格每行右侧出现删除按钮
- 可删除整个领域或单条阶段目标
- 删除前弹出确认弹窗

**添加目标弹窗**
- 字段：年龄段及领域（文本）、干预目标（文本）、阶段目标（多行文本，每行一个）
- 若领域已存在则追加目标，否则新建领域行

**导出 Word**
- 生成 HTML 表格 → Blob 下载为 .doc 文件
- 表格样式：蓝色表头、宋体12pt、边框合并

### 新旧功能关系

IEP页，可以沿用现有的界面功能
基于评估结果推导出来IEP，也是能支持编辑修改
