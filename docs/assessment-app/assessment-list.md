### 功能说明

**左侧导航栏**
- 学生头像 + 姓名 + 性别·年龄
- 导航项：康复概述、评估报告（选中）、1对1训练课、小组课、康复报告、残联报告
- 底部"冻结"链接

**Tab 筛选**
- 全部 / 已完成 / 评估中，选中态蓝色下划线
- 右侧"新建评估"蓝色文字按钮

**评估记录卡片**
- 标题行：日期 + 状态（评估中=蓝色，已完成=绿色）
- 信息行：评估时间、工具名称、评估师
- 操作按钮：评估中→【删除】+【继续评估】，已完成→【查看】

**新建评估弹窗**
- 弹出模态弹窗，展示可用评估工具列表（详见界面8：选择评估工具弹窗）

**测试数据生成按钮**
- 页面右侧固定浮动，3 个按钮：🟢高分 / 🟡中分 / 🔴低分
- 点击后为所有已启用工具各生成一条已完成记录 + 答题数据

| 档位 | 最高分概率 | 中间分概率 | 最低分概率 |
|------|-----------|-----------|-----------|
| 高分 | 80-100%   | 0-20%     | 0%        |
| 中分 | 40-60%    | 20-40%    | 20%       |
| 低分 | 10%       | 10-20%    | 60-80%    |

**删除确认弹窗**
- 自定义弹窗（非浏览器原生），确认后删除记录和答题数据

### 新旧功能关系

这是已有界面，在评估记录里面，新增字段【评估工具】

## 界面8：选择评估工具弹窗（弹窗，位于 assessment-list.html）

### 界面路径

```
评估记录列表页
  └─ [新建评估] ──→ 选择评估工具弹窗 ← 当前界面
```

### 功能说明

**弹窗触发**
- 点击列表页右上角"新建评估"按钮弹出

**工具卡片列表**
- 仅展示 `status=active` 的工具
- 每张卡片展示：工具图标（40px 圆角方形）、名称、描述、题目数、预估用时
- 图标颜色：PEP-3=紫色底、VB-MAPP=蓝色底、C-PEP=默认
- 自定义工具若有上传头像则展示头像

**交互行为**
- hover：边框变蓝 + 浅蓝背景
- 点击工具卡片 → 创建新评估记录（status=ongoing）→ 跳转评估工具介绍页
- 点击 × 或遮罩层关闭弹窗
