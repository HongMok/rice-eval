### 功能说明

**Tab 结构**
- Tab1: 评估报告 | Tab2: IEP | Tab3: 评估量表

**Tab1 — 评估报告内容模块**

| 模块 | 说明 |
|------|------|
| 基本信息 | 姓名、出生日期、实足年龄、发展年龄、评估时间、评估师 |
| 功能发展评分表 | P/E/F 体系，7 个领域，简要评价列（得分率≥75%=强处🟢，≤40%=弱处🔴） |
| 行为表现评分 | A/M/S 体系，4 个领域，pill 按钮展示评级 |
| 副测验分数表 | 各副测验原始分→发展年龄→百分比级数→发展/适应程度 |
| 合成分数表 | 沟通(C)=CV+CP+I、运动(Mo)=FM+GM+EH、适应行为=R+M+S+L |
| 发展表现图 | 折线图，X轴=副测验，Y轴=发展/适应程度等级 |
| 雷达图 | 7 个功能发展领域得分率可视化 |
| 综合分析 | 自动生成，支持编辑保存 |

> 报告计算规则详见：`config/report-prompt-pep3.md`

**Tab2 — IEP（个别化教育计划）**
- 详见界面9：IEP 个别化教育计划
- IEP 生成规则详见：`config/iep-prompt-pep3.md`

**Tab3 — 评估量表**
- 详见界面10：评估量表详情

**底部操作栏**
- 【导出PDF】：调用 `window.print()`
