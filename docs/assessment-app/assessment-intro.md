### 功能说明

**页面入口**
- 从列表页新建评估弹窗选择工具后跳转到此页
- URL 参数：`?record={recordId}&tool={toolId}`

**工具基本信息卡片**
- 工具图标 + 名称
- 类型标签、题目数量、预估用时、适用年龄

**富文本介绍区域**
- 内容来源：后台编辑页配置的 `intro` 字段（HTML 富文本）
- 支持标题、段落、列表、加粗等格式
- 若未配置介绍内容，显示默认占位文字
- 数据来源优先级：后台配置的 `intro` 字段 → 内置默认介绍文档

**底部操作栏**
- 固定底部，白色背景
- 居中蓝色实心按钮"开始评估"
- 点击后跳转到答题页：`assessment-answer.html?record={recordId}&tool={toolId}`
