### 功能说明

**Tab1 — 评估报告内容模块**

| 模块 | 说明 |
|------|------|
| 基本信息 | 姓名、出生日期、实足年龄、评估时间、评估师 |
| 各领域得分表 | 7 个领域：通过/中间反应/不通过数 + 得分 + 发展年龄 + 得分率进度条 |
| 记分册 | 功能发展量表 + 行为病理量表，双栏展示各领域通过数/总分 |
| 发展功能量表侧面图 | 折线图，X轴=7个领域，Y轴=发展年龄 |
| 行为病理行为靶心图 | 同心圆靶心图，3环（适当A/轻度M/严重S），各行为领域标注在对应环上 |
| 雷达图 | 7 个领域得分率可视化 |
| 综合分析 | 自动生成，支持编辑保存 |

> 报告计算规则详见：`config/report-prompt-cpep.md`

**Tab2 — IEP**
- 详见界面9：IEP 个别化教育计划
- IEP 生成规则详见：`config/iep-prompt-cpep.md`

**Tab3 — 评估量表**
- 详见界面10：评估量表详情
- 左侧导航：7 个领域直接列出（无分组层级）
- 评分选项：不通过(0) / 中间反应(0.5) / 通过(1)
