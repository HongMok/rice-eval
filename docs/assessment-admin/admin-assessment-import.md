### 功能说明

**三步流程**

| 步骤 | 说明 |
|------|------|
| ① 下载模板 | 提供 PEP-3 和 VB-MAPP 两套预设模板下载 |
| ② 上传文件 | 拖拽或点击上传，支持 .xlsx / .csv |
| ③ 预览数据 | 解析后展示预览表格，异常行红色高亮 |

**CSV 表头格式**

| 列名 | 必填 | 说明 |
|------|------|------|
| 排序 | 否 | 缺失时自动递增 |
| 一级分类 | **是** | 分组名称（group） |
| 二级分类 | **是** | 子测验/领域名称（domain） |
| 评估项目 | 否 | 题目简短名称 |
| 操作描述 | **是** | 题目详细描述 |
| 所需材料 | 否 | 评估所需教具 |
| 适用年龄 | 否 | 如 12-24月 |
| 是否必答 | 否 | 默认"是" |
| 评分_{字母}_{字段} | 动态 | 字母编号 a/b/c...，字段: 分值/标签/说明 |

**解析规则**

```
1. 读取表头行，识别所有列索引
2. 正则 ^评分_([a-z])_(分值|标签|说明)$ 匹配评分列
3. 按字母排序评分列组（a, b, c...），构建 scoreColMap
4. 逐行解析：
   - 缺少一级分类或二级分类 → 标记无效，跳过
   - 构建 domainKey = {一级分类}__{二级分类}
   - 每行生成 Question 对象，scores 数组从评分列动态构建
5. 返回：config（按 domainKey 分组）、totalCount、invalidCount
```

**导入结果**
- 隐藏三步流程，显示：✅ 导入完成，成功 N 条，失败 N 条
- 底部按钮变为"返回编辑页"

**CSV 文件规范**
- 所有字段 QUOTE_ALL（双引号包裹）
- 双引号转义：两个双引号
- UTF-8 编码，不带 BOM
- Excel (.xlsx) 使用 SheetJS 库解析
