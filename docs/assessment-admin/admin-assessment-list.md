### 功能说明

**卡片网格**
- 3列网格布局，每张卡片展示：圆形图标、名称、描述（2行截断）、toggle 开关、题数/修改时间、编辑链接
- 停用卡片 opacity: 0.55，hover 恢复 0.85
- 末尾固定虚线卡片"+ 新建评估工具"

**筛选条件区**
- 左侧：名称搜索框 + 状态下拉（全部/启用/停用，onchange 实时筛选）+ 搜索按钮 + 重置按钮
- 右侧：「类型管理」按钮

**启用/停用规则**

| 规则 | 说明 |
|------|------|
| 默认状态 | 新建工具默认"停用" |
| 启用校验 | 题目数=0 时禁止启用，提示"该评估工具尚无题目，无法启用"，开关自动恢复 |
| 切换方式 | 点击 toggle 直接生效，无确认弹窗 |
| 停用影响 | 停用工具在康复师 App 新建评估弹窗中不可见 |

**类型显示名映射**

| type | 显示名 |
|------|--------|
| pep3 | PEP-3 · 心理教育评估 |
| cpep | C-PEP · 孤独症儿童发展评估 |
| vbmapp | VB-MAPP · 语言行为里程碑评估 |
| common | 通用评估 · 综合型评估 |

### 新旧功能关系
- **在现有系统，左侧导航栏，新增【评估】按钮，点击进入评估工具管理**
- **左侧导航栏，沿用之前其他的tab的逻辑，双层左侧目录。第二级目录默认收起即可**

## 界面3：新建评估工具弹窗（弹窗，位于 admin-assessment-list.html）

### 界面路径

```
评估工具列表页
  └─ [+ 新建评估工具] ──→ 新建评估工具弹窗 ← 当前界面
```

### 线框图

```
+--------------------------------------------------+
|  新建评估工具                              [×]    |
|  请先填写基本信息，然后选择配置方式                 |
|                                                    |
|  +------+  名称 *: [________________]              |
|  | 📋   |  类型 *: [请选择评估工具类型 ▼]           |
|  | 上传  |  描述:   [____________________]          |
|  +------+          [____________________]          |
|                                                    |
|  选择配置方式:                                      |
|  +--------------------+ +--------------------+     |
|  | 📥 表格录入         | | ✏️ 手动录入         |     |
|  | 下载模板，填写评估   | | 在线逐条添加评估    |     |
|  | 题目后上传导入       | | 领域和题目          |     |
|  +--------------------+ +--------------------+     |
+--------------------------------------------------+
```

### 功能说明

**基本信息**
- 图标上传（64px 圆形）、名称（必填）、类型下拉（必填）、描述
- 类型下拉为自定义组件（非原生 select），每项展示名称+描述
- 选择类型后，若描述为空则自动填入该类型的工具简介

**表单校验规则**

| 校验 | 提示 |
|------|------|
| 名称为空 | "请输入评估工具名称" |
| 类型未选 | "请选择评估工具类型" |
| 名称重复 | "评估工具名称"XXX"已存在，请使用其他名称" |

**配置方式**
- 手动录入：创建工具（status=inactive, questionCount=0）→ 跳转空白编辑页
- 表格导入：弹窗内切换为导入步骤（隐藏基本信息区），展示三步导入流程
