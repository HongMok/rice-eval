### 功能说明

**基本信息区**
- 默认只读展示：头像 + 名称 + 类型 + 描述 + 评估工具介绍状态（"已配置"/"未配置"）
- 点击"编辑"弹出右侧抽屉（520px），含：图标上传、名称、类型（只读）、描述、富文本介绍编辑器
- 富文本编辑器：contenteditable div，工具栏含 B/I/U/H/OL/UL，数据存储为 `intro` 字段（HTML 字符串）
- 抽屉底部"删除此评估工具"红色链接：必须先停用才能删除

**课包干预目标关联栏**
- 位置：基本信息区和题目列表之间
- 横向排列所有课包图标（48px 紫色圆形 + 文字缩写）+ 课包名称
- 已启用：正常显示 + 右上角绿色勾 ✓；未启用：灰度显示
- 点击任意课包图标 → 进入全屏干预目标关联界面
- 数据源：`config/course-packets.json`，RICE1v1 默认关闭（需人为打开）

**左侧领域结构树**
- 宽度 280px，两级结构：分组 → 子测验/领域
- 普通模式：点击分组折叠/展开，点击子项选中加载右侧题目
- 编辑模式：节点 hover 显示 ✏️ 改名 / 🗑 删除，底部"+ 添加分组"/"+ 添加子组"

**右侧题目表格**
- 列：排序号（可排序）、评估项目、操作描述、所需材料、适用年龄、是否必答、启用状态、评分等级（动态列）、操作（编辑/删除）
- 题目编辑抽屉（520px）：核心字段（排序号、项目名、描述、评分说明）+ 扩展字段（材料、年龄、必答、启用，默认折叠）

**评分体系规则**

```
新增题目评分来源优先级：
1. 同领域已有题目 → 复制其 scores 结构（分值+标签，说明留空）
2. 领域内无题目 → fallback 到工具类型预设评分体系

预设评分体系：
  PEP-3:   2-通过 / 1-萌芽 / 0-不通过
  C-PEP:   1-通过 / 0.5-中间反应 / 0-不通过
  VB-MAPP: 1-1分 / 0.5-0.5分 / 0-0分
```

### 功能说明

**顶部栏**
- 返回按钮、课包名称、关联备注标签、匹配进度（X/Y 已匹配，仅预设模式）、启用/关闭 toggle

**模式选择**（两张卡片，单选）

| 模式 | 说明 |
|------|------|
| 预设匹配 | 每个评估项目和课包内干预项目做匹配，可手动编辑或AI自动匹配 |
| 规则匹配 | 不做预设匹配，提供匹配规则文本，AI在生成IEP时自动匹配 |

切换模式时，若当前模式有数据则弹窗确认。

**预设匹配 — 子模式**（radio pills）

| 子模式 | 说明 |
|--------|------|
| 评估项目匹配 | 每个评估项目关联多个干预目标（一对多） |
| 评估分数匹配 | 同一评估项目下，不同评分等级分别匹配不同干预目标 |

切换子模式时，若有匹配数据则弹窗确认（切换会清空）。

**左侧目录树**
- 分组 → 子领域两层结构，与编辑页树一致
- 每个子领域显示匹配进度（如 `3/10`）
- 颜色：全匹配=绿色，部分=紫色，无=灰色

**右侧详情面板**
- 评估项目匹配模式：每行显示题号、评估项目、匹配的干预目标（名称+领域标签）、操作
- 评估分数匹配模式：主行显示评估项目+配置状态，子行按评分等级展开

**配置状态判断**

```
已配置（绿色）：所有评分等级都配置了干预目标
部分配置（橙色）：部分评分等级配置了
未配置（灰色）：无任何配置
```

**AI自动匹配**
- 点击弹确认（已有数据时提示会清空）
- 按领域关键词智能匹配，匹配完成后所有评估项目都有对应干预目标

**导出关联表**
- CSV 文件，9列：评估项目题号、评估项目、评估项目描述、评估项目评分、评估项目评分标签、关联干预项目ID、关联干预项目、关联干预项目大领域、关联干预项目子领域
- 评估分数匹配模式下同一评估项目按评分等级展开多行
- UTF-8 BOM 编码，QUOTE_ALL

### 功能说明

**搜索**
- 支持多关键词空格分隔（AND 逻辑），最大50字符
- 实时搜索，输入框右侧出现 × 清除按钮
- 清除后回到 AI 推荐列表

**AI推荐**
- 默认展示，基于评估项目领域智能推荐
- 顶部紫色"☆ AI推荐"标签 + 右侧"刷新"按钮

**干预目标列表**
- 每项显示：紫色一级领域标签 + 橙色二级细分领域标签 + 项目名称
- 支持多选（toggle 选中/取消）
- 已选项以紫色标签展示（带 × 删除按钮）

**干预目标数据结构（3层）**

```
territory（一级领域）→ subdomain（二级细分领域）→ name（项目名称）

示例：
  动作发展 → 粗大动作 → 多种基本姿势
  动作发展 → 精细动作 → 手指精细操作
  沟通     → 语言理解 → 听觉注意
  沟通     → 语言表达 → 仿说
  认知     → 配对     → 视觉配对
  社交情绪 → 模仿     → 动作模仿
```
