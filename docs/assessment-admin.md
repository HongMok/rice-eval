## 系统界面跳转关系
```plaintext
评估工具列表页 (admin-assessment-list.html)
  │
  ├─ [类型管理] ──→ 类型配置管理弹窗（Modal，页内弹出）
  │
  ├─ [+ 新建评估工具] ──→ 新建评估工具弹窗（Modal，页内弹出）
  │     ├─ [手动录入] ──→ 评估工具编辑页（空白）
  │     └─ [表格导入] ──→ 弹窗内切换为导入步骤
  │           └─ [导入完成] ──→ 评估工具编辑页
  │
  ├─ [卡片"编辑"] ──→ 评估工具编辑页 (admin-assessment-edit.html?id=xxx)
  │     │
  │     ├─ [< 返回] ──→ 评估工具列表页
  │     │
  │     ├─ [基本信息"编辑"] ──→ 基本信息编辑抽屉（Drawer，页内滑出）
  │     │
  │     ├─ [题目"编辑"/"+添加题目"] ──→ 评估项目编辑抽屉（Drawer，页内滑出）
  │     │
  │     ├─ [课包图标] ──→ 课包干预目标关联 — 全屏界面（新页面）
  │     │     │
  │     │     ├─ [预设匹配 → 设置/编辑] ──→ 匹配设置抽屉（Drawer）
  │     │     │
  │     │     └─ [规则匹配 → 编辑] ──→ 规则编辑弹窗（Modal）
  │     │
  │     └─ [导入入口] ──→ 批量导入页 (admin-assessment-import.html?id=xxx)
  │           └─ [< 返回编辑页] ──→ 评估工具编辑页
  │
  └─ [toggle 开关] ──→ 直接切换启用/停用（无跳转）

```

### 功能说明

**基本信息区**

- 默认只读展示：头像 + 名称 + 类型 + 描述 + 评估工具介绍状态（"已配置"/"未配置"）
- 点击"编辑"弹出右侧抽屉（520px），含：图标上传、名称、类型（只读）、描述、富文本介绍编辑器
- 富文本编辑器：contenteditable div，工具栏含 B/I/U/H/OL/UL，数据存储为 `intro` 字段（HTML 字符串）
- 抽屉底部"删除此评估工具"红色链接：必须先停用才能删除

**课包干预目标关联栏**

- 位置：基本信息区和题目列表之间
- 横向排列所有课包图标（48px 紫色圆形 + 文字缩写）+ 课包名称
- 已启用：正常显示 + 右上角绿色勾 ✓；未启用：灰度显示
- 点击任意课包图标 → 进入全屏干预目标关联界面
- 数据源：`config/course-packets.json`，RICE1v1 默认关闭（需人为打开）

**左侧领域结构树**

- 宽度 280px，两级结构：分组 → 子测验/领域
- 普通模式：点击分组折叠/展开，点击子项选中加载右侧题目
- 编辑模式：节点 hover 显示 ✏️ 改名 / 🗑 删除，底部"+ 添加分组"/"+ 添加子组"

**右侧题目表格**

- 列：排序号（可排序）、评估项目、操作描述、所需材料、适用年龄、是否必答、启用状态、评分等级（动态列）、操作（编辑/删除）
- 题目编辑抽屉（520px）：核心字段（排序号、项目名、描述、评分说明）+ 扩展字段（材料、年龄、必答、启用，默认折叠）

**评分体系规则**
```plaintext
新增题目评分来源优先级：
1. 同领域已有题目 → 复制其 scores 结构（分值+标签，说明留空）
2. 领域内无题目 → fallback 到工具类型预设评分体系

预设评分体系：
  PEP-3:   2-通过 / 1-萌芽 / 0-不通过
  C-PEP:   1-通过 / 0.5-中间反应 / 0-不通过
  VB-MAPP: 1-1分 / 0.5-0.5分 / 0-0分

```

```plaintext

4. 逐行解析：
   - 缺少一级分类或二级分类 → 标记无效，跳过
```
